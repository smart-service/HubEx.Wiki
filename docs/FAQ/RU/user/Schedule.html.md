## Графики работы в HubEx

Графики работы позволяют:

* Учитывать рабочее время сотрудников при назначении на заявки;
* Учитывать переработки сотрудников при выполнении заявок;
* Автоматически назначать исполнителей на заявки;
* Просматривать время работы объекта обслуживания;
* Настраивать SLA заявок с учетом времени обслуживания.

Подробнее с правилами автоназначения исполнителей на заявки вы можете ознакомиться в статье: [Правила автоназначения исполнителя](https://wiki.hubex.ru/docs/FAQ/RU/admin/RulesOfChoice.html.md)

## Графики работы по-умолчанию

По-умолчанию в бизнес-аккаунте HubEx созданы следующие графики работы:

* Выходные 9-18;
* 2/2(4) c 9:00 до 18:00;
* 2/2(3) с 9:00 до 18:00;
* 2/2(2) c 9:00 до 18:00;
* 2/2 c 9:00 до 18:00;
* Понедельник-Пятница 9:00-18:00;
* Круглосуточный 24/7.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule23.png)

**Обратите внимание!** В HubEx нет возможности отобразить в графиках работы обеденное время.

## Как создавать или редактировать графики работы

Создание и редактирование персональных графиков находится в [Консоли администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.md) в разделе левого бокового меню **"Структура предприятия"** - подраздел **"Графики работы"**.

Создать новый график можно по кнопке **"Создать график"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule.jpg)

В системе предусмотрено создание и настройка двух типов графиков:

* Недельный: включает в себя настройку рабочего времени на каждый из семи дней недели. Такой тип графика подойдет для создания, например, пятидневной или шестидневной рабочей недели с двумя или одним выходным днем.
* Сменный: подразумевает создание повторяющегося интервала смен и выходных дней. Этот тип подходит для создания таких графиков работы, как 2/2,1/3 и т.д.

Рассмотрим создание каждого типа графиков подробно.

## Создание недельного графика работы

Для создания недельного графика выполните следующие шаги:

1. Введите **Наименование**: для удобства наименование следует задавать исходя из типа графика и времени работы. Например, для графика работы со вторника по субботу с 10.00 до 19.00 и выходными в воскресенье, и понедельник можно ввести наименование Вторник-суббота с 10 до 19;
2. С помощью флажков укажите, какие дни недели будут являться рабочими по графику, и установите время работы в формате чч:мм. Для текущего примера с графиком работы со вторника по субботу с 10.00 до 19.00 и двумя выходными в воскресенье и понедельник необходимо установить флажки для вторника, среды, четверга, пятницы, субботы, а для воскресенья и понедельника флажки не ставить, так как это нерабочие дни;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule2.jpg)

3. Установите дату начала действия текущего графика с помощью флажка **Назначить дату начала календаря**: этот пункт не является обязательным для выполнения, если дату начала не задать, то по умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать с 20 февраля 2024 года;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule22.jpg)

4. Проверьте отображение графика по кнопке **Показать график в календаре**: в новом окне появится распределение графика работы по месяцам. При внесении изменений в график кнопку **Показать график в календаре** нужно нажимать повторно;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule3.jpg)

5. Нажмите кнопку **"Создать график"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule4.jpg)

Чтобы использовать автоназначение на заявки по графику и отслеживать рабочее время исполнителей по календарю, необходимо указать созданный график в карточке **Сотрудника**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule5.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule6.jpg)

## Создание сменного графика работы

Для создания сменного графика выполните следующие шаги:

1. Введите **Наименование**: для удобства наименование следует задавать исходя из типа графика и времени работы. Например, для сменного графика работы два дня рабочих с 8:30 до 20:00 и два дня выходных, можно ввести наименование "2/2 с 8:30 до 20:00";
2. По-умолчанию для настройки сменного графика на форме представлен интервал 2/2: две рабочих смены с 8:30 до 20:00 и два выходных дня. Такой интервал будет повторяться друг за другом. Чтобы изменить настройки по умолчанию удаляйте строки смен и (или) выходных дней и добавляйте новые по кнопке + **Добавить**. Для каждой строки можно изменить тип дня графика: **Смена** или **Выходной**.

   Для нашего примера графика 2/2 поправим только время начала и окончания смен с 8.30 до 20.00.

   Обратите внимание! Время одной смены не может быть более 24 часов. Если ваши сотрудники работают более 24 часов в смене, создайте подряд две рабочие смены на 24 часа и на дополнительное время.

   Продолжительность смены рассчитывается исходя из установленного времени и отображается под каждой строкой смены.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule7.png)
3. Установите дату начала действия текущего графика с помощью флажка **"Назначить дату начала календаря"**: этот пункт не является обязательным для выполнения, если дату начала не задать, то по умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать с 23 мая 2024 года;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule8.png)
4. Проверьте отображение графика по кнопке **"Показать график в календаре"**: в новом окне появится распределение графика работы по месяцам. При внесении изменений в график кнопку **"Показать график в календаре"** нужно нажимать повторно;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule9.png)
5. Нажмите кнопку **"Создать график"**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule10.png)

Рассмотрим еще один пример создания сменных графиков: создание сменного графика 3/2/3, где первые 3 дня это дневные смены с 6:00 до 16:00, следом 2 дня ночные смены с 16:00 до 2:00 и 3 выходных дня.

Для создания графика 3/2/3 выполните следующие шаги:

1. Введите **Наименование**: 3/2/3 с 6 до 16, с 16 до 2;
2. Как описано выше, по-умолчанию для настройки сменного графика на форме представлен интервал 2/2. Интервал по умолчанию необходимо изменить под требования примера 3/2/3 следующим образом:
   * В первой строке для типа дня **Смена** измените количество на 3 и установить время начала 6:00 и время окончания смены на 16:00;

     ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule11.png)
   * Так как следующая часть интервала - это ночные смены, то для второй строки интервала с типом дня **Выходной** измените тип на **Смена**, задайте количество 2 и установите время с 16:00 до 2:00.

     Обратите внимание! При задании ночных смен вводите время окончания следующего дня (например, 2 часа ночи), система автоматически рассчитает время продолжительности смены с переходом на следующий день и отобразит значение под строкой смены;

     ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule12.png)
   * По кнопке **"+Добавить"** согласно графику добавьте строку с типом **Выходной** и количеством 3;
3. Установите дату начала действия текущего графика с помощью флажка **"Назначить дату начала календаря"**: этот пункт не является обязательным для выполнения, если дату начала не задать, то по-умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать с 23 мая 2024 года;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule13.png)
4. Проверьте отображение графика по кнопке **Показать график в календаре**: в новом окне появится распределение графика работы повторяющегося интервала 3/2/3;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule14.png)
5. Нажмите кнопку **"Создать график"**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule15.png)

## Графики работы для сотрудников

**График работы** позволят распределять **Заявки** на исполнителей точно по времени. Вы увидите рабочие и выходные дни сотрудников.

**График работы** выбирается для каждого **Сотрудника** индивидуально в его карточке на вкладке **Трудоустройство**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/CreatingUser/Employment.jpg)

Как использовать **График работы**? В **Заявке** при выборе **Исполнителя** укажите специалиста, которого хотите назначить. Вы увидите его расписание на текущий день: рабочий день (белые поля), выходной (заштрихованные поля), назначенные на него **Заявки** и продолжительность их выполнения. Эта информация позволит вам сделать выбор при назначении - оценить свободное время, выбрать другой рабочий день этого специалиста, либо выбрать другого специалиста.

Выбрать другой день можно с помощью стрелок вперед-назад.

Для назначения **Заявки** выберите подходящее время начала выполнения и кликните на него левой кнопкой мыши.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Schedule/SelectEngeneer.jpg)

Обратите внимание! После того как вы установите **Заявку** на подходящее время на форме выбора **Исполнителя** параметры **Дата и время выполнения** установятся автоматически: **Начало** - та дата и время, на которое вы установили Заявку, **Окончание** - высчитывается исходя из установленных **Плановых трудозатрат** в настройке **Вида работ**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Schedule/WorkType.jpg)

С помощью графика работы исполнителей вы можете посмотреть отработанное время исполнителей и их переработки в отчете по сотрудникам. Подробнее читайте в статье: [Отработанное время](https://wiki.hubex.ru/docs/FAQ/RU/user/EngineersAnalytics.html.md#second)

## Как изменить график сотрудника

В WEB-версии HubEx можно отредактировать смены можно двумя способами.

1. **Первый способ**: в карточке **Сотрудника** (левое боковое меню **Пользователи** - подраздел **"Сотрудники"**) на вкладке **"Трудоустройство"**. На вкладке **"Трудоустройство"** вы увидите уже заданный график работ. Подробнее читайте в статье: [График работы исполнителей](https://wiki.hubex.ru/docs/FAQ/RU/user/Schedule.html.md).

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/Schedule.jpg)

   **Обратите внимание!** Смены можно устанавливать, даже если в карточке сотрудника НЕ выбран график работ.

   Нажмите кнопку **"Редактировать смену"** и выберите необходимую дату для детальной настройки:

   * Если вы выбрали рабочий день, то вы можете отредактировать время выхода на смену, либо сделать день нерабочим, выбрав вариант **Не на смене**.

     После внесения изменений нажмите кнопку **Сохранить**;

     ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/EdittingSchedule.jpg)
   * Если вы выбрали выходной день, то вы можете сделать день рабочим, выбрав вариант **На смене** и указать время смены;

     После внесения изменений нажмите кнопку **Сохранить**.

     ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/EdittingSchedule2.jpg)
2. **Второй способ**: в расписании заявок (раздел левого бокового меню **"Заявки"** - подраздел **"Расписание заявок"**) при работе в любом из режимов представления (день, неделя, месяц) у каждого **Сотрудника** в списке отображается индикатор смены: зеленый значок – на смене в текущее время, серый значок – не на смене. Для редактирования смены необходимо нажать на контекстное меню (три точки) и выбрать пункт **"Редактировать смену"**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/Calendar.jpg)

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/Calendar2.jpg)

   Выберите необходимую дату для детальной настройки:

   * если вы выбрали рабочий день, то вы можете отредактировать время выхода на смену, либо сделать день нерабочим, выбрав вариант **Не на смене**.

     После внесения изменений нажмите кнопку **"Сохранить"**;
   * Если вы выбрали выходной день, то вы можете сделать день рабочим, выбрав вариант **На смене** и указать время смены;

     После внесения изменений нажмите кнопку **Сохранить**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/Calendar3.jpg)

Отредактировать заданный график работы исполнитель может самостоятельно в мобильном приложении **HubEx исполнитель**.

**Обратите внимание!** Смены можно устанавливать, даже если в карточке сотрудника НЕ выбран график работ.

Откройте боковое меню и нажмите на иконку исполнителя.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob.jpg)

Перед вами откроется заданный график работы. Напоминаем, что график работы устанавливается в WEB-версии в карточке **Сотрудника** на вкладке **"Трудоустройство"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob2.jpg)

Выберите необходимую дату для детальной настройки:

* Если вы выбрали рабочий день, то вы можете отредактировать время выхода на смену, либо сделать день нерабочим, выбрав вариант **"Не на смене"**.

  После внесения изменений нажмите кнопку **Сохранить**;

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob3.jpg)
* Если вы выбрали выходной день, то вы можете сделать день рабочим, выбрав вариант **"На смене"** и указать время смены;

  После внесения изменений нажмите кнопку **"Сохранить"**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob4.jpg)

## Графики работы для объектов

За любым **Объектом/оборудованием** можно закрепить **График работы**. Поле будет носить исключительно справочную информацию.

* Перейдите в раздел левого бокового меню **"Объекты/оборудование"**;
* Переместитесь в карточку **Объекта/оборудования**;
* В поле **График работы** добавьте график работы объекта/оборудования.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ScheduleNew/ScheduleNew1.png)

## Графики работы в SLA

Также **Графики работы** используются в [SLA - Правила расчета крайнего срока закрытия заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/SLA.html.md).

**График обслуживания заявки** определяет рабочее время выполнения заявок. Например, если вы установили в правило SLA график работы "Понедельник-пятница с 9:00 до 18:00" и плановое время закрытия заявки 45 часов, далее создали заявку в понедельник в 9:00, то крайний срок закрытия этой заявки будет определен на пятницу в 18:00.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ScheduleNew/ScheduleNew2.png)

## Функционал "Я на смене"

Диспетчеру сервисной службы или начальнику сервиса удобно видеть сотрудников, которые работают в данный момент времени, чтобы оперативно назначать на них **Заявки**.

Сервисному специалисту также важно предупреждать работодателя о том, что он приступил к работе, чтобы получать **Заявки**, либо закончил смену и не может больше брать **Заявки**.

В "Мобильном приложении HubEx для исполнителя" перейдите в левое боковое меню. Для того чтобы начать смену, нажмите кнопку **"Начать смену"**.

**Обратите внимание!** Вы НЕ можете указать другое время начала смены. Поле **"Время по"** заполняется автоматически, исходя из настроек графика.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob5.jpg)

Если вы хотите уменьшить или увеличить количество часов в смене, то сначала измените ее, и только потом нажимайте кнопку **"Начать смену"**.

По завершению рабочего дня перейдите также в левое боковое меню и нажмите кнопку **"Завершить смену"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob6.jpg)

По завершению смены вы увидите расчет времени смены и нажмите кнопку **"Сохранить"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/OnDuty/ScheduleMob7.jpg)
