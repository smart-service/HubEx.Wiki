#### Фильтр по сотрудникам в календаре: обновление от 17.01.2024

Работать с назначением заявок теперь можно еще более эффективно! Встречайте полезное обновление: возможность фильтрации по сотрудникам в расписании заявок (календаре).

С помощью календаря можно оперативно планировать расписание исполнителей. А благодаря новому функционалу работа пользователей в больших компаниях станет еще удобнее.

##### Фильтры по сотрудникам в календаре

Фильтры по сотрудникам доступны для дневного, недельного и месячного отображения календаря. С помощью фильтров можно по различным критериям отобрать необходимых исполнителей для назначения **Заявок**.

Рассмотрим, как это работает. Перейдите в меню **Заявки - Расписание заявок** и выберите необходимое отображение календаря.

Над списком сотрудников слева находится кнопка **Фильтровать сотрудников**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterInCalendar/CalendarFilterEmployee.jpg)

По нажатию на кнопку фильтров справа откроется боковое окно с параметрами доступными для выбора сотрудников. Отбор можно осуществлять по **Видам работ**, **Участкам**, **Компаниям**, **Ролям** и признаку **Мобильный сотрудник**.

Обратите внимание! Признак **Мобильный сотрудник** по умолчанию включен в настройках фильтров, так как это признак выездных сотрудников, на которых назначаются **Заявки**. Чтобы увидеть в списке других сотрудников, снимите флажок в настройках фильтров.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterInCalendar/CalendarFilterEmployee2.jpg)

В модальном окне для каждого из параметров доступен множественный выбор. Например, можно выбрать несколько **Видов работ** для поиска подходящих сотрудников и нажать кнопку **Выбрать**.

Таким образом, можно более точно отфильтровать подходящих исполнителей. Фильтрация по сотрудникам также удобна при работе в группе компаний, когда сотрудники устроены в разные компании одного холдинга, когда сотрудников очень много.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterInCalendar/CalendarFilterEmployee4.jpg)

После указания всех параметров на боковой панели фильтров нажмите кнопку **Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterInCalendar/CalendarFilterEmployee5.jpg)

Список сотрудников будет отфильтрован согласно заданным фильтрам, также сетка календаря будет скорректирована согласно отбору. После применения фильтра по сотрудникам над списком сотрудников будет отображаться количество подходящих исполнителей, а также кнопка сброса фильтрации.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterInCalendar/CalendarFilterEmployee6.jpg)

Таким образом новый функционал позволит диспетчерам, руководителям подразделения эффективнее работать с расписанием заявок: подбирать исполнителей по **Видам работ**, отслеживать загрузку сотрудников в разрезе **Компаний** и **Участков**, быстро и гибко принимать решения о назначении **Заявок** на сотрудников с определенными ролями в вашей компании.
