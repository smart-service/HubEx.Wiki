#### Справочники в дополнительных полях: обновление от 16.05.2025

# Системные справочники в дополнительных полях

**Дата релиза:** 16 мая 2025
 **Раздел:** Заявки / Администрирование / Дополнительные поля

---

## Что реализовано

Добавлена возможность использования системных справочников в дополнительных полях для заявок. Теперь в системе **HubEx** можно создавать дополнительные поля с типом **«Значение из справочника»** с поддержкой как одиночного, так и множественного выбора значений. Новые поля позволяют работать с ключевыми сущностями системы напрямую из карточки заявки.

---

## Что нового

### Поддержка системных справочников в дополнительных полях

При создании дополнительного поля в админ панели доступен новый тип:

* **Значение из справочника** — выбор значения из одной из системных сущностей:

  + Объекты/Оборудование
  + Компании
  + Сотрудники
  + Заказчики
  + Материалы

Дополнительно можно включить множественный выбор через соответствующий чек-бокс «**Разрешить выбирать несколько значений»**

![image](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DirectoriesInAdditionalFields16052025/QWKS2vP_image.png)

---

### Использование в заявках

В карточке заявки теперь можно использовать поля со справочниками.

Справочники используют уже существующие списки элементов системы — объектов, сотрудников, компаний и т.д.

Отображение данных по справочникам:

* **Объекты** — полное наименование с учётом иерархии
* **Компании** — название компании
* **Сотрудники** и **Заказчики** — ФИО, должность, аватар
* **Материалы** — наименование материала

![image](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DirectoriesInAdditionalFields16052025/twK9317B_image.jpg)

---

### Обновления админ панели по дополнительным полям

В меню **Настройки системы → Дополнительные поля** обновлён интерфейс создания и редактирования:

* Добавлен новый тип поля: **Значение из справочника**
* Убрана опция **«Множественный выбор из списка»** — множественный выбор теперь задаётся через чек-бокс «**Разрешить выбирать несколько значений»**
* Для типа «**Значение из справочника**» можно выбрать нужный справочник и разрешить множественный выбор

---

## Важно

* На текущий момент функциональность справочников в дополнительных полях доступна **только для добавления в заявки**.
   Поддержка других сущностей (объекты, компании и др.) будет реализована в последующих обновлениях.
