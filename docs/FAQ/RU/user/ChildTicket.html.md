#### Дочерние заявки

В этом разделе вы узнаете:

* [Как создать дочернюю заявку](https://wiki.hubex.ru/docs/FAQ/RU/user/ChildTicket.html.md#ChildTicket1);
* [Как из любой заявки сделать дочернюю](https://wiki.hubex.ru/docs/FAQ/RU/user/ChildTicket.html.md#ChildTicket2);
* [Как отслеживать дочерние заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/ChildTicket.html.md#ChildTicket3);
* [Что делать, если дочерних заявок не видно](https://wiki.hubex.ru/docs/FAQ/RU/user/ChildTicket.html.md#ChildTicket4).

**Заявки** в системе могут иметь иерархическую структуру: дочерние заявки подчиняются родительским.
Создание дочерних заявок будет полезным в случаях:

* необходимости разбить работы на отдельные задачи в рамках объемного проекта;
* необходимости назначить нескольких специалистов на работу с объектом: в таком случае для каждого исполнителя
  создается своя дочерняя заявка;
* объединения задач от заказчика в одно задание.

Дочерние заявки не отличаются от обычных заявок. Они имеют такие же поля для заполнения.

##### Создание дочерних заявок

Создать **Дочернюю заявку** можно несколькими способами:

1. На вкладке **Дочерние заявки** в **Заявке**.
   Также здесь отображаются уже ранее созданные дочерние заявки.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/ChildTicket.jpg)

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/ChildTickets.jpg)

   По кнопке **Создать дочернюю заявку** открывается краткая форма полей заявки, в которой можно задать отличного от
   родительской заявки **Исполнителя**, **Вид работ**, **Тип заявки**, назначить другое
   время выполнения. Обратите внимание! При
   таком способе создания дочерней заявки в краткой форме нельзя выбрать/изменить **Объект**. **Объект** будет
   наследоваться
   от родительской
   заявки. Изменить **Объект** можно после сохранения дочерней заявки: кликните на необходимую **Заявку** и в боковом окне измените **Объект**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/NewChildTicket.jpg)

   Полная форма **Дочерней заявки** выглядит точно так же как и простая заявка. Отличительной чертой **Дочерней заявки**
   является заполненное поле **Родительская заявка**. В нем отражается номер **Родительской заявки**.

   По щелчку на любую **Заявку** из списка дочерних она откроется в боковой панели справа.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/SelectChildTicket.jpg)

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/OpenedChildTicket.jpg)

Боковое окно содержит основные поля из полной формы **Заявки**, их можно редактировать и сохранять изменения. Кнопка **Сохранить** находится внизу боковой формы **Заявки**. По нажатию на кнопку **К заявке** в соседней вкладке будет открыта полная форма текущей **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/OpenedChildTicket.jpg)

2. Любую заявку можно превратить в дочернюю. Для этого в **Заявке** на вкладке **Заявка** нужно заполнить
   поле **Родительская
   заявка** и нажать кнопку **Сохранить**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/ParentTicket.jpg)

##### Статистика дочерних заявок

В общем списке заявок у каждой карточки заявки будут отражаться родительские и дочерние заявки, которые представлены
гиперссылками: щелчок мыши по ссылке **Родительской заявки** откроет форму родительской заявки; щелчок мыши по ссылке
**Дочерние заявки** откроет родительскую заявку с вкладкой **Дочерние заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/ChildParentTickets.jpg)

В табличном отображении списка заявок также есть ссылки на родительские и дочерние заявки. Настроить отображение
полей для списка в табличном формате можно по кнопке **Настройка** (синяя шестеренка).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/TableTickets.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/TableTickets2.jpg)

Добавленные столбцы можно будет перемещать по таблице. Подробнее читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/docs/FAQ/RU/user/Filters.html.md).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/TableTickets3.jpg)

##### Дочерние заявки не отражаются внутри родительской

В карточке **Заявки** на вкладке **Дочерние заявки** может возникнуть ситуация: не отображаются дочерние заявки, хотя их
количество видно и в списке заявок и на самой вкладке. Чтобы это исправить нажмите кнопку **Настройки** (синяя
шестеренка).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/Settings.jpg)

В настройках полей выберите необходимые, либо установите флажок в пункте **Выбрать все** для выбора всех полей. Таким
образом дочерние заявки будут отображаться в стандартном режиме.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/Settings2.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChildTicket/Settings3.jpg)
