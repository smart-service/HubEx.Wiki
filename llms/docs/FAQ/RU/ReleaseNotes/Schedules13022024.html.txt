#### Создание и настройка недельных графиков работы: обновление от 13.02.2024

Команда HubEx представляет гибкий функционал по созданию и настройке пользовательских графиков работы. Теперь диспетчер или руководитель может создать любой недельный график для своих сотрудников в соответствии с потребностями бизнеса. В скоре к функционалу также добавится создание сменных графиков!

##### Графики работы

**График работы** исполнителей позволяет учитывать рабочее время сотрудников при назначении на **Заявки**. Также **График работы** можно использовать при настройке автораспределения исполнителей на **Заявки**.

Создание и редактирование графиков доступно в консоли администрирования в меню **Структура предприятия - Графики работы** по кнопке **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule.jpg)

Недельный график включает в себя настройку рабочего времени на каждый из семи дней недели. Такой тип графика подойдет для создания, например, пятидневной или шестидневной рабочей недели с двумя или одним выходным днем.

Для недельного графика рабочие дни задаются с помощью установки флажков и временного интервала.

Также для графика можно задать дату начала действия графика с помощью параметра **Назначить дату начала календаря**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule22.jpg)

Посмотреть распределение графика по календарю можно по кнопке **Показать график в календаре**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule3.jpg)

Для сохранения графика в системе необходимо нажать кнопку **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule4.jpg)

Чтобы использовать автоназначение на **Заявки** по графику и отслеживать рабочее время исполнителей по календарю, необходимо указать созданный график в карточке **Сотрудника**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule5.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Schedules/Schedule6.jpg)

Подробнее читайте в статье нашей базы знаний: [Настройка графиков работы](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/WorkSchedule.html.txt).
