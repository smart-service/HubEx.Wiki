#### График обслуживания объектов

**График обслуживания объектов** - это форма представления **Плановых заявок**. Он
используется при работе с
планово-предупредительными работами, техническим обслуживанием объектов.

График позволяет увидеть картину запланированных работ в разрезе оборудования и периода, а также сокращает время работы
исполнителей, работающих с **Плановыми заявками**. Подробнее читайте в статье: [Создание плановой заявки](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/PlannedTickets.html.txt).

Прочтите подробную статью ниже или начните знакомство с темой с обучающего видеоролика **График обслуживания**.

VK
YouTube

График находится в меню **Заявки - Планировщик работ**. С помощью переключателя можно выбрать режим
работы либо с
**Плановыми заявками**, либо с **Графиком обслуживания объектов**. При работе с графиком кнопка **Создать заявку** также будет доступна для создания **Плановой заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Menu.jpg)

В графике отображаются **Заявки**, созданные через планировщик за выбранный период времени.

На панели параметров график необходимо настроить:

* **Тип заявки**: выбор **Тип заявки** - обязательное условие для формирования графика,
  можно выбрать как один, так и несколько типов;
* **Вид работ**: также позволяет уточнить построение графика для одного или нескольких **Видов
  работ**, указанных в плановых Заявках;
* **Заказчик**: выбор **Заказчика** - одно из условий для формирования **Графика
  обслуживания объектов**. **График обслуживания объектов** может быть сформирован как для одного, так и для нескольких **Заказчиков**. **Заказчик** выбирается из списка **Компаний**,
  в карточке которых выбран тип **Заказчик**
  или **Наша компания**. В списке будут доступны только те **Заказчики**, по которым были созданы **Плановые заявки**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Customer.jpg)

* в полях **Тип объекта** и **Класс объекта** можно уточнить параметры для дальнейшего
  выбора **Объектов** для построения графика. Также класс и тип могут участвовать в группировке
  графика (задать группировку можно в параметре **Группировка объектов по**);
* **Объект**: вы можете выбрать один или несколько **Объектов** компании-заказчика, по
  которым хотите сформировать график. Обратите внимание, выбрать можно только родительский объект. При выборе
  родительского объекта
  автоматически будут выбраны все его дочерние объекты;
* в полях **Период С - По** необходимо задать период, на который нужно отобразить график
  обслуживания;
* в параметре **Группировка периода по** можно указать одно из трех значений:
  + **по дням** - график будет представлен в виде таблицы по каждому **Объекту** с
    детализацией по
    количеству **Заявок** на каждый день выбранного периода;

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Day.jpg)

  + **неделям** - график будет представлен в виде таблицы по каждому **Объекту** с
    детализацией по
    количеству **Заявок** на каждую неделю выбранного периода. В заголовке столбца указана дата начала недели;

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Week.jpg)

  + **месяцам** - график будет представлен в виде таблицы по каждому **Объекту** с
    детализацией по
    количеству **Заявок** на каждый месяц выбранного периода;

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Month.jpg)
* в поле **Группировка объектов по** можно задать группировку **Объектов** в таблице по классам, типу или
  оставить группировку по умолчанию.

Обратите внимание! Для формирования **Графика обслуживания объектов** после заполнения всех параметров необходимо нажать кнопку
**Просмотр отчета**. При изменении любых настроек графика кнопку **Просмотр отчета** нужно
нажимать повторно.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Button.jpg)

Готовый график можно распечатать с помощью меню **Файл - Печать**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Print.jpg)

Также график можно экспортировать в различные форматы - pdf, xlsx, docx, xml и другие. Для последующего
редактирования файла рекомендуется выбирать формат xlsx.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTicketsSchedule/Export.jpg)
