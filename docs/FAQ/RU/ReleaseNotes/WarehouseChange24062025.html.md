#### Обновление в HubEx - Теперь можно менять склад в Израсходованных материалах выполненной работы!

Мы добавили возможность **смены Склада** в поле "**Израсходованные материалы**" выполненной работы в **Заявке**.

Теперь вы можете **менять Склад напрямую**, без необходимости повторного подбора материалов. Это особенно полезно, если исполнитель по **Заявке** поменялся после заполнения выполненной работы и требуется изменить **Склад**.

Раньше для этого нужно было полностью удалять все материалы и выбирать их заново — это занимало время и повышало риск ошибок. Теперь достаточно просто изменить **Склад** у уже добавленного материала.

**Как это работает:**

1. Перейдите в Заявку, где есть выполненные работы.
2. Откройте блок "**Израсходованные материалы**".
3. Найдите нужный материал и кликните по полю **Склад** рядом с ним — откроется выпадающий список.

   ![Смена склада у израсходованного материала](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/WarehouseChange24062025/GfqZ977y_Screenshot-2025-06-25-at-21-07-10.png)
4. В списке отобразятся только те **Склады**, на которых доступен данный материал. Выберите нужный.
5. Если на других **Складах** такого материала нет, то в списке будет доступен только текущий **Склад**.
6. Нажмите кнопку **"Сохранить"**, чтобы применить изменения.
