#### Вход под учётной записью “Владельца тенанта” и обычного пользователя тенанта

Тенант - это рабочая область в HubEx. Владелец тенанта имеет две учетные записи в системе:

1. Первая запись обозначается при входе как **Имя\_пользователя (Системный администратор)** и имеет следующие
   особенности:
   * ограничения по ролям не распространяются: пользователю доступны всевозможные данные и объекты;
   * пользователь видит все удаленные **Объекты**, **Заявки** и т.д.
   * для этого пользователя невозможно настроить видимость полей на форме **Заявки** - они все доступны по
     умолчанию, скрыть их нельзя;
   * данный пользователь может переводить **Заявку** на любую стадию из жизненного цикла выбранного **Типа заявки**,
     в том числе и
     на стадии, которые не связаны между собой.Учетную запись владельца тенанта можно использовать для настройки системы, для установления ограничений с
   помощью настройки **Ролей** и т.д.
2. Вторая запись обозначается как **Имя\_пользователя**: эта учетная запись будет иметь роль **Системный администратор**,
   которую уже можно настроить аналогично с другими **Ролями** в системе. Подробнее читайте в статье: [Настройка ролей](https://wiki.hubex.ru/docs/FAQ/RU/admin/Roles.html.md).
