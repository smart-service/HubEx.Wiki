#### Полезные обновления (релевантность, паспорт объекта, экспорт): обновление от 28.08.2023

Три полезных обновления HubEx для улучшения работы пользователей:

* отображение критериев релевантности при подборе исполнителей;
* отображение ссылок в паспорте объекта;
* фиксация последних настроек экспорта заявок для пользователя.

Рассмотрим подробнее каждую из полезных функций текущей версии.

##### Релевантность в подборе исполнителей

Выбор исполнителей в **Заявках** по релевантности - это стратегическое преимущество для вашего бизнеса. Оценивая соответствие сотрудника для выполнения задачи, вы обеспечиваете эффективное исполнение **Заявок** ваших клиентов.

Поэтому мы расширяем и улучшаем возможности данного функционала.

В текущем обновлении вас ждет отображение релевантности для каждого сотрудника.

Как это работает?

При выборе исполнителя в **Заявке** по релевантности для каждого сотрудника будут отображаться иконки критериев отбора (справа от ФИО).

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ReleasePack28082023/1.png)

При наведении на иконки критериев всплывает подсказка с расшифровкой обозначений.

Чем больше критериев отбора совпадает, тем сотрудник выше в списке по релевантности.

Таким образом, вы сможете принимать решения о назначении того или иного сотрудника, опираясь на критерии отбора.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ReleasePack28082023/2.png)

Подробнее читайте в статье нашей базы знаний: [Создание заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingTicket.html.md).

##### Ссылки в паспорте объекта

Паспорт объекта - это данные объекта (оборудования), которые отражаются в мобильном приложении **HubEx исполнитель** и **HubEx заказчик** при сканировании QR-кода. Также паспорт объекта отображается при сканировании QR-кода через камеру телефона незарегистрированными в системе пользователями.

Теперь в паспорте объекта все ссылки кликабельны. Паспорт объекта отображается согласно полям, заполненным в карточке **Объекта**. Так, например, если в карточке **Объекта** в поле **Описание** будет ссылка на инструкцию, то ее можно будет открыть прямиком из паспорта.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ReleasePack28082023/3.png)

Подробнее читайте в статье нашей базы знаний: [Создание шаблона заявки, подача заявки по QR-коду: паспорт объекта](https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingTaskTemplates.html.md#passportl).

##### Фиксация настроек экспорта заявок

Благодаря экспорту, **Заявки** из HubEx можно выгружать в таблицу Excel для дальнейшей загрузки в сторонние системы.

Теперь для каждого пользователя последние настройки экспорта заявок фиксируются: запоминается вариант экспорта (Обычный экспорт, Расширенный экспорт), а также выбор полей для расширенного экспорта.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ReleasePack28082023/4.png)

Обратите внимание! Настройки фиксируются для каждого браузера и компьютера индивидуально.

Подробнее читайте в статье нашей базы знаний: [Экспорт](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md).

Предложения по доработкам в системе и сообщения об ошибках просим отправлять к нам в "чат с поддержкой" в веб-приложении или на почту  [help@hubex.ru](mailto:help@hubex.ru) .
