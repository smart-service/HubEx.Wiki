#### Дополнительные поля для компаний и договоров: обновление от 19.12.2023

Управлять данными по компаниям и контрактами в HubEx теперь можно более эффективно! Встречайте полезное обновление: возможность создавать дополнительные поля для компаний и договоров.

Дополнительные поля расширяют возможности хранения учетной информации в системе. Например, вы можете создать поля для нормативных требований по договору, прикладывать инструкции, указывать сроки, для компаний можно создавать условия сотрудничества, чтобы опираться на них при заключении новых сделок.

##### Дополнительные поля для компаний

Дополнительные поля создаются в консоли администратора в меню **Настройки системы - Дополнительные поля** по кнопке **Создать поле**:

* **Название поля** внесите в свободной форме. Например, для компании можно создать варианты условий сотрудничества, чтобы указывать разные наборы условий для разных компаний;
* **Тип поля**: выбирается под назначение данных. Например, подходящий тип для создания условий - **Множественный выбор из списка**.

  Вы можете создавать строковые поля и указывать, например, комментарии, дату основания компании и другие поля для хранения полезной информации;
* **Единицу измерения** можно выбрать согласно тому, в каких единицах вы хотите указывать информацию (единицы указываются для измеримых величин);
* В поле **Используется для** нужно указать использование для **Компаний**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsContract/AddFieldCompany0.jpg)

Все дополнительные поля будут располагаться в карточке компании внизу под **Контактными лицами**.

Обратите внимание! Созданные дополнительные поля будут доступны для каждой компании уже созданной в системе и каждой новой компании.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsContract/AddFieldCompany.jpg)

##### Дополнительные поля для договоров

Дополнительные поля для договоров создаются также в консоли администратора в меню **Настройки системы - Дополнительные поля** по кнопке **Создать поле**:

* **Название поля** внесите в свободной форме. Например, для создания переключателя о подписании договора в названии можно указать - Договор подписан;
* **Тип поля**: выбирается под назначение данных. Например, подходящий тип для отображения статуса договора - **Переключатель**.

  Вы можете создавать строковые поля и указывать, например, условия взаиморасчетов, создавать числовые поля и указывать максимальную стоимость сделки и так далее;
* **Единицу измерения** можно выбрать согласно тому, в каких единицах вы хотите указывать информацию (единицы указываются для измеримых величин);
* В поле **Используется для** нужно указать использование для **Договоров**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsContract/AddField.jpg)

Все дополнительные поля будут располагаться в карточке договора внизу. Все договоры хранятся в разрезе заказчиков в карточке компании на вкладке **Договоры**.

Обратите внимание! Созданные дополнительные поля будут доступны для каждого существующего в системе договора и каждого нового договора.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsContract/AddField2.jpg)

Таким образом, создание дополнительных полей для компаний и контрактов позволит вам удобно хранить всю необходимую информацию в одном месте и эффективно управлять данными.

Подробнее читайте в статье нашей базы знаний: [Дополнительные поля](https://wiki.hubex.ru/docs/FAQ/RU/user/AdditionalFields.html.txt).
