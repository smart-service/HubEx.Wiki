#### Расположение Объекта на плане по умолчанию: обновление от 13.03.2024

Встречайте новое полезное обновление для HubEx: для **Объекта** на плане теперь можно установить расположение по умолчанию. Местоположение по умолчанию будет отображаться на плане в **Заявке**, а значит позволит выездным специалистам легко находить **Объект**, экономя время и уменьшая путаницу.

##### Расположение объекта на плане по умолчанию

Чтобы установить расположение по умолчанию перейдите в меню **Объекты|Оборудование - Объекты|Оборудование**и откройте необходимый **Объект**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FloorPlan/Object.jpg)

Нажмите на план в карточке **Объекта** и в открывшемся окне установите расположение этого **Объекта** на плане, щелкнув левой кнопкой мыши. На выбранном месте отобразится пин (булавка). После выбора местоположения нажмите **Сохранить** или **Сохранить и закрыть**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FloorPlan/PlanInObject.jpg)

Расположение по умолчанию будет отображаться на плане при подаче **Заявки** по **Объекту**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FloorPlan/PlanInTicket.jpg)

Если **Объект** на момент подачи **Заявки** располагался в другом месте, то на плане можно выбрать другую локацию.

Обратите внимание! При изменении местоположения на плане в самой **Заявке** расположение по умолчанию в карточке **Объекта** изменено НЕ будет.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FloorPlan/Plan.jpg)

Подробнее читайте в статье нашей базы знаний: [Планы помещений](https://wiki.hubex.ru/docs/FAQ/RU/user/FloorPlan.html.md).
