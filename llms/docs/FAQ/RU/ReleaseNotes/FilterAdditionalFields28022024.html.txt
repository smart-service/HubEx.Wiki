#### Фильтрация заявок по дополнительным полям со списочным форматом: обновление от 28.02.2024

Команда HubEx предлагает новые возможности для работы с дополнительными полями! Теперь дополнительные поля формата даты и списков доступны для фильтрации **Заявок**.

Пользователи легко смогут найти необходимые **Заявки**, используя фильтры по значениям из дополнительных полей.

Это позволит оптимизировать использование дополнительных полей и более эффективно находить необходимую информацию.

##### Фильтрация заявок по дополнительным полям

Вы уже знаете, что в HubEx список **Заявок** можно фильтровать, используя текстовые, числовые и дополнительные поля формата диапазона данных. Теперь в рамках этого функционала для поиска доступны дополнительные поля с форматами:

* **Дата** и **Дата и время**;
* **Значение из списка** и **Множественный выбор из списка**.

Дополнительные поля создаются и настраиваются в консоли администратора. Подробнее об этом читайте в статье: [Создание дополнительных полей для заявки](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/AdditionalFields.html.txt)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterList/AddFieldList.jpg)

Дополнительные поля находятся на форме **Заявки** внизу списка и могут быть доступны как пользователям web-приложения, так и мобильных приложений **HubEx исполнитель** и **HubEx заказчик**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterList/Ticket.jpg)В меню **Заявки - Все заявки** на панели фильтров внизу будут доступны дополнительные поля для отбора.

Для списочных форматов дополнительных полей доступен выбор значений, соответствующих типу списка - единичный выбор или множественный выбор. Для формата дат и времени будет доступен календарь для выбора дат и счетчик времени. Таким образом вы сможете быстро и эффективно находить **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterList/FilterList.jpg)

Выберите необходимые значения поиска для нажмите кнопку **Применить** в правом верхнем углу.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterList/ListsFilter.jpg)

Список **Заявок** будет отфильтрован согласно установленным параметрам поиска по дополнительным полям.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterList/ResultFilter.jpg)

Дополнительно о работе с фильтрами для **Заявок** читайте в статье нашей базы знаний: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/Filters.html.txt).

Предложения по доработкам в системе и сообщения об ошибках просим отправлять к нам в "чат с поддержкой" в веб-приложении или на почту  [help@hubex.ru](mailto:help@hubex.ru) .
