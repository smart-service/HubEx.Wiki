## **🚀 Управление правами для массового редактирования от 28.07.2025**

## 🌟 Что нового?

Мы добавили более гибкую систему управления правами для массовых операций с заявками и объектами. Теперь администраторы могут точнее настраивать доступ к функциям редактирования.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ManageRightsForBulkEditing28072025/HTRVW6Xn_1.jpg)

## 🔐 Ключевые изменения

### Для заявок:

* Добавлен новый пермишен **"Массовое редактирование заявок"** для массового редактирования
* Если право отсутствует - пункт "Редактировать" скрывается в меню массовых операций (при выборе нескольких заявок)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ManageRightsForBulkEditing28072025/Y781cvDp_2.jpg)

* Сохранена текущая логика:
  + Без права на изменение заявок недоступно как единичное, так и массовое редактирование
  + С правом на изменение, но без "**Массовое редактирование заявок**" - доступно только единичное редактирование

### Для объектов:

* Добавлен новый пермишен **"Массовое редактирование объекта"** для массового редактирования
* Если право отсутствует - пункт "Редактировать" скрывается в меню массовых операций (при выборе нескольких объектов)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ManageRightsForBulkEditing28072025/k2XgVwmG_3.jpg)

* Сохранение текущего поведения:
  + Без права на изменение объектов недоступно никакое редактирование
  + С правом на изменение, но без "**Массовое редактирование объекта**" - доступно только единичное редактирование

## 🛠️ Как это работает?

### Сценарий 1: Полные права

* Пользователь имеет:
  + Право на изменение заявок/объектов
  + Право на массовое редактирование
* Результат:
  + Доступно и единичное, и массовое редактирование

### Сценарий 2: Только единичное редактирование

* Пользователь имеет:
  + Право на изменение заявок/объектов
  + Нет права на массовое редактирование
* Результат:
  + Доступно только единичное редактирование
  + Пункт "Редактировать" в массовых операциях скрыт

### Сценарий 3: Нет прав

* Пользователь не имеет:
  + Права на изменение заявок/объектов
  + Права на массовое редактирование
* Результат:
  + Все функции редактирования заблокированы

## ⚙️ Преимущества новой системы

* **Гибкость** - более точная настройка прав для разных ролей
* **Прозрачность** - четкие правила доступности функций
