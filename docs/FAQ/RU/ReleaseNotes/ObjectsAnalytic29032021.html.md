#### Отчет по объектам: обновление от 29.03.2021

В меню **Аналитика** теперь доступен **Отчет по объектам**. Аналитическая панель по объектам обслуживания содержит в себе множество полезных показателей и диаграмм.

С помощью **Отчета по объектам** вы получаете данные по работе с объектами в разных плоскостях: количество объектов в системе, активные объекты, а также среднее время выполнения Заявок по объектам, сколько проходит времени между Заявками и т.д.

Показатели средних значений отображаются наверху аналитической панели, для каждого показателя есть всплывающая информационная подсказка. Для каждого гафика и диаграммы также есть всплывающие подсказки-помощники.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsAnalytic29032021/1.png)

С помощью панели фильтров вы можете сделать детальную настройку для показателей и диаграмм по **Объектам**, **Типам заявки** и т.д.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsAnalytic29032021/2.png)

Внизу аналитической панели находится **Таблица показателей** по всем объектам (в иерахическом представлении).

Здесь можно оценить, например, время между подачей **Заявок**: как часто приходят **Заявки** по объекту обслуживания? С этой информацией вы запросто сможете решить вопрос - а в чем же причина частых **Заявок**? Некачественный сервис? Или оборудование пора заменить, а не тратить время на постоянные ремонты?

А почему такие большие сроки закрытия **Заявок**? Может быть специалисты долго добираются? Значит нужно оптимизировать назначение специалистов в засимости от удаленности от **Заявки**. Или исполнители просто забывают закрывать задачи? Может после завершения работ проще автоматически закрывать **Заявки** с помощью настройки жизненного цикла?

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsAnalytic29032021/3.png)

Вы можете выбрать конкретный объект в таблице и открыть по нему детальный отчет по кнопке В отчет по объекту. Здесь будет доступна аналитическая информация только по выбранному объекту: сколько закрытых **Заявок**, сколько на период открытых, **Заявки** в разрезе исполнителей и т.д.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsAnalytic29032021/4.png)

Подробнее об отчете читайте в статье базы знаний: [Отчет по объектам](https://wiki.hubex.ru/docs/FAQ/RU/user/ObjectsAnalytics.html.md).
