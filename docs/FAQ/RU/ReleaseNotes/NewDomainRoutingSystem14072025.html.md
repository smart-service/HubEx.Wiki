## 🚀 Новая система маршрутизации по доменам - от 14.07.2025

---

## 🌟 Что нового?

Мы внедрили улучшенную систему маршрутизации для более надежной работы платформы HubEx, без необходимости использования отдельного мобильного приложения! Теперь доступна возможность указания домена для маршрутизации на нужный облачный сервер

## 📌 Основные изменения

### 1. Новая логика авторизации

* На экране входа появилось поле **"Указать домен"**
* Доступно для всех типов входа:
  + По номеру телефона
  + По email
  + Через SSO

### 2. Работа с доменами

* Пользователь вводит домен организации (например: "api.example.ru")
* Система направляет запросы на соответствующий сервер HubEx
* При ошибке подключения выводится уведомление

### 3. Обработка ошибок

* Если домен неверный или сервер недоступен:
  + Показывается сообщение об ошибке
  + Предлагается проверить правильность ввода
  + Возможность повторить попытку

## 🛠️ Как это работает?

### 🔄 Стандартный сценарий

1. Пользователь открывает приложение
2. Вводит логин (email/телефон)
3. Указывает домен организации
4. Система направляет запрос на соответствующий сервер
5. Происходит авторизация

## 🏢 Что такое выделенный сервер HubEx?

Для корпоративных клиентов мы предлагаем возможность размещения HubEx на **отдельном облачном сервере**. Это означает, что ваша компания получает:

* **Выделенные ресурсы** – сервер работает исключительно для вашей организации
* **Повышенную производительность** – нет конкуренции за ресурсы с другими клиентами
* **Гибкие настройки** – возможность адаптировать инфраструктуру под специфические требования
* **Дополнительную безопасность** – изолированное окружение для ваших данных

Благодаря новой системе маршрутизации, пользователи вашей компании смогут автоматически попадать на **ваш выделенный сервер HubEx**, просто указав корпоративный домен при авторизации. Это особенно важно для:
✔ Крупных предприятий с большим числом пользователей
✔ Компаний с особыми требованиями к безопасности
✔ Организаций, нуждающихся в кастомизированных решениях

Если вашей компании требуется выделенный сервер – обратитесь к вашему менеджеру в HubEx для обсуждения условий подключения.
