#### Фильтрация заявок по дополнительным полям: обновление от 23.09.2022

Дополнительные поля расширяют возможности работы с **Заявками**: в них можно указывать площадь объектов, вносить заключение по итогу работы, указывать дату диагностикии т.д. Подробнее читайте в статье нашей базы знаний: [Создание дополнительных полей для заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/FloorPlan.html.txt).

Теперь искать **Заявки** в общем списке можно, используя отбор по дополнителым полям.

Как это работает?

С помощью фильтров добавьте отбор по дополнительным полям.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketsFilter23092022/wLMtcbt_Screenshot-15.png)

Выберите необходимые поля: они могут быть в формате даты, чисел, строки, значений из списка.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketsFilter23092022/85rddjB_Screenshot-16.png)

Задайте значения фильтров для отбора **Заявок** по дополнительным полям. Будут отобраны те **Заявки**, для которых совпадает отбор хотя бы по одному из значений всех указанных параметров.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketsFilter23092022/bLF7nB7_Screenshot-17.png)

Предложения по доработкам в системе и сообщениям об ошибках просим отправлять к нам в "чат с поддержкой" в веб-приложении или на почту  [help@hubex.ru](mailto:help@hubex.ru) .
