#### Мобильное оборудование: заявка по мобильному оборудованию

Мобильное оборудование - это оборудование с динамичным адресом. Новый адрес такого оборудования указывается в **Заявке** и автоматически сохраняется в карточке
**Объекта**.

Мобильным оборудованием может быть строительная, шахтовая техника, которая перемещается по разным локациям, или,
например, вендинговые автоматы. Также к мобильному оборудованию можно отнести оборудование, которое сдается в аренду.

Разберем подробнее, как это работает.

Мобильным оборудованием может являться только объект или оборудование с **Типом оборудования**, для
которого в настройках указан флажок **Обязательное наличие адреса (при создании объекта оборудования)**.
Это родительские объекты. Дочерние объекты наследуют адреса от родительских. Поэтому если дочерние объекты будут
являться мобильными их адрес не будет изменяться, так как он наследуется от родителя.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/MobileObjects/Type.jpg)

Флажок **Мобильное оборудование** устанавливается в карточке **Объекта**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/MobileObjects/Object.jpg)

Для мобильного оборудования при подаче **Заявки** есть возможность изменить адрес и сохранить его для
выбранного **Объекта**:

* для поля **Адрес** нажмите кнопку **Редактировать -
  Изменить местоположение** и укажите новый адрес;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/MobileObjects/Ticket.jpg)

* сохраните **Заявку** по кнопке **Сохранить** или **Сохранить и закрыть**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/MobileObjects/Ticket2.jpg)

* откройте карточку **Объекта**: его адрес изменится на тот, который вы указали в **Заявке**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/MobileObjects/Object2.jpg)

Таким образом, каждый раз указывая новый адрес в **Заявке** в карточке **Объекта** этот адрес будет обновляться.

Посмотреть историю перемещений такого оборудования вы можете с помощью экспорта **Заявок** с отбором по этому оборудованию. Подробнее читайте в статье: [Экспорт](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.txt).

Обратите внимание! Если вы создадите две **Заявки** на мобильное оборудование и измените адрес только у второй по счету **Заявки**, то в первой **Заявке** адрес не изменится. В первой **Заявке** адрес нужно будет изменить вручную, если это необходимо.
