<h3><strong>✅ Редактор чек-листов удобный выбор объектов с поддержкой иерархии</strong></h3>
<p class="ds-markdown-paragraph">Мы обновили механизм добавления объектов в чек-листы. Процесс стал более гибким и визуально понятным &mdash; теперь он работает по принципу привычного фильтра по заявкам.</p>
<h4>🔄&nbsp;<strong>Что было раньше?</strong></h4>
<p>При создании или редактировании чек-листа выбор объектов происходил в устаревшей модальном окне, которое не позволяла быстро выбирать вложенную структуру родительских объектов. Это создавало неудобства, когда нужно было выбрать группу объектов вместе с их дочерними элементами.</p>
<h4>✨&nbsp;<strong>Что изменилось теперь?</strong></h4>
<p>Мы полностью заменили интерфейс выбора объектов:</p>
<ul>
<li>
<p class="ds-markdown-paragraph"><strong>Новая модалка &laquo;Поиск по объектам&raquo;:</strong>&nbsp;В ней отображаются все доступные объекты с полной иерархией. Вы можете&nbsp;заходить в&nbsp;группы&nbsp;и выбирать как конкретные элементы, так и все дочерние элементы.</p>
</li>
<li>
<p class="ds-markdown-paragraph"><strong>Понятный сбор данных:</strong>&nbsp;Справа в модальном окне формируется список выбранного, как в фильтрах по заявкам. Вы всегда видите итоговый набор перед подтверждением.</p>
</li>
</ul>

<h4>🛠️&nbsp;<strong>Пример работы</strong></h4>
<p>Вы создаете чек-лист для технического обслуживания и выбираете в модальном окне:</p>
<ol start="1">
<li>
<p class="ds-markdown-paragraph">Отмечаете&nbsp;оборудование&nbsp;&laquo;Цех №1&raquo; (родитель).</p>
</li>
<li>Выбираете дочерние элементы&nbsp;&laquo;Цех №1&raquo;;</li>
<li>
<p class="ds-markdown-paragraph">Дополнительно отмечаете конкретный станок &laquo;Токарный станок ЧПУ-5&raquo; из другого цеха;</p>
</li>
</ol>
<p class="ds-markdown-paragraph">После нажатия &laquo;Выбрать&raquo; система:</p>
<ul>
<li>
<p class="ds-markdown-paragraph">Добавит все дочерние объекты для &laquo;Цех №1&raquo;, в том числе сам объект &laquo;Цех №1&raquo;.</p>
</li>
<li>
<p class="ds-markdown-paragraph">Добавит объект&nbsp;&laquo;Токарный станок ЧПУ-5&raquo;.</p>
</li>
</ul>
<hr>
<h3><strong>✅ Плагины в HubEx: теперь доступны прямо из верхнего меню</strong></h3>
<p class="ds-markdown-paragraph">Мы сделали работу со сторонними интеграциями еще удобнее. Плагины теперь можно добавлять прямо в верхнее меню&nbsp;панели диспетчера&nbsp;&mdash; во всплывающем окне.</p>
<h4>🚀&nbsp;<strong>Что нового?</strong></h4>
<p>В правой части верхнего строки панели диспетчера есть возможность добавить плагин, который отображается в виде отдельных кнопок.</p>
<h4>🖱️&nbsp;<strong>Как это работает?</strong></h4>
<ul>
<li>
<p class="ds-markdown-paragraph"><strong>Быстрый доступ:</strong>&nbsp;Рядом с вашим профилем отображаются кнопки плагинов с иконками. Просто нажмите на нужную, чтобы открыть окно с плагином.</p>
</li>
<li>
<p class="ds-markdown-paragraph"><strong>Несколько плагинов:</strong>&nbsp;Система поддерживает добавление нескольких плагинов. Если плагинов больше трех, остальные автоматически сворачиваются в меню &laquo;Еще&raquo;.</p>
</li>
<li>
<p class="ds-markdown-paragraph"><strong>Подсказки:</strong>&nbsp;При наведении на иконку&nbsp;отображается название плагина.</p>
</li>
</ul>
<h4>🪟&nbsp;<strong>Удобное окно с плагином</strong></h4>
<ul>
<li>
<p class="ds-markdown-paragraph">Плагин открывается во всплывающем окне прямо под кнопкой.</p>
</li>
<li>
<p class="ds-markdown-paragraph">В заголовке отображается название плагина.</p>
</li>
</ul>
<h4>🔐&nbsp;<strong>Безопасно и прозрачно</strong></h4>
<p>Плагин открывается с вашими правами доступа: в него автоматически передается актуальный токен авторизации.</p>
<h4>🛠️&nbsp;<strong>Технические данные</strong></h4>
<p>Для добавления плагина в верхнее меню диспетчера требуется указывать значение "ResourceID":33.</p>
