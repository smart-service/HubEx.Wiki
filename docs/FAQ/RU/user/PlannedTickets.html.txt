**Заявки** и
уведомлять ваших

##### Создание Плановой заявки в web-приложении

Создать **Плановую заявку** можно в меню **Заявки - Планировщик работ** по кнопке **Создать
заявку**. Форма создания **Плановой заявки** состоит из нескольких разделов:
**Название**, **Тип
заявки**, **Вид работ**, **Объект/оборудование**, **Описание**
**Исполнитель**, **Плановое создание заявок**, раздел **Ограничить
повторения** (раздел появляется после выбора **Частоты создания заявок**).

Процесс заполнения полей **Плановой заявки** похож на заполнение стандартной **Заявки** в HubEx.
Подробнее читайте в статье: [Создание заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingTicket.html.txt#webticket).

Рассмотрим подробнее заполнение полей **Плановой заявки**:

* **Название** заполняется в свободной форме;
* с указанным **Типом заявки** будут создаваться **Плановые заявки**. **Тип
  заявки** может быть выбран только один;
* **Вид работ** может быть выбран только один. По **Виду работ** подбираются **Объекты/оборудование**
  для создания **Плановой
  заявки** (**Виды работ** заполняются в карточке **Объекта**). На следующем
  шаге вы можете уточнить выбор **Объектов**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/PlannedTicket0.jpg)

* **Плановые заявки** можно подавать не только на один **Объект/оборудование**, но и на
  группу **Объектов** с указанным выше **Видом работ**. После выбора **Вида
  работ** станут активными кнопки **К списку объектов** для выбора **Объектов** вручную, а также кнопка **Подобрать объекты** для установки дополнительных фильтров для подбора **Объектов**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/PlannedTicket1.jpg)

  Нажмите на кнопку **К списку объектов** и установите флажки на **Объектах**,
  которые вам нужны для
  создания **Заявок**, затем нажмите кнопку **Выбрать**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/ListObj.jpg)

  Если необходимо, то можно уточнить выбор **Объектов** с помощью фильтров по кнопке **Подобрать объекты**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/FilterObj.jpg)
* в поле **Описание** можно внести описание заявки в свободной форме;
* в разделе **Плановое создание заявок** настраивается частота создания **Плановых
  заявок**. В
  выпадающем списке
  выберите один из вариантов: **Ежедневно**, **Еженедельно** и т.д.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/plantick3.png)

  Для каждого из вариантов будет доступна детальная настройка времени. Например, необходимо каждые 3 недели по
  вторникам создавать **Плановую заявку** в 10 часов утра.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/TimeManager.jpg)
* параметр **Отложить начало выполнения работ** позволит скорректировать срок закрытия заявки.
  Например, **Плановая
  заявка** будет создана в 12:00 и плановый срок закрытия заявки 4 часа. Если данный параметр не
  указан, то срок по
  заявке будет 16:00. Если же данный параметр равен 2 часа, то срок будет определен как 18:00.
* в разделе **Ограничить повторения** вы можете указать **Период создания заявки**,
  например, **Плановые заявки** вам нужно
  создавать в течение года. Либо ограничить повторения по количеству, например, вам нужно всего 10 заявок.
* в поле **Исполнитель** вы можете выбрать одного или нескольких исполнителей на выполнение **Плановой заявки**. Для этого нажмите кнопку **+Добавить исполнителя**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/SelectEngineer.jpg)

  С помощью установки флажков выберите исполнителей и нажмите кнопку **Выбрать**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/SelectEngineer2.jpg)

  Для **Плановой заявки** можно установить переключатель **Автоматическое назначение исполнителя**. Это означает, что исполнитель будет подбираться при создании
  заявки по
  специальным
  правилам автоназначения исполнителя.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/SelectEngineer3.jpg)

  **Правило автоназначения исполнителя** можно создать в [консоли
  администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.txt) в меню Настройки заявки - Автоназначение исполнителя. Подробнее читайте в статье: [Правила автоназначения исполнителя](https://wiki.hubex.ru/docs/FAQ/RU/admin/RulesOfChoice.html.txt).

  После создания правила его необходимо привязать к стадии заявки **Новая**, чтобы исполнитель
  назначался
  автоматически именно при создании **Плановой заявки**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/Stage.jpg)

  Обратите внимание! Внося изменения в стадию заявки эти изменения будут работать для всех **Типов
  заявок**, в
  жизненный цикл которых включена эта стадия. Вы можете создать отдельный **Тип заявки** для
  **Плановой заявки**,
  создать новую стадию Новая (для плановой заявки), настроить жизненный цикл. И выбрать такой тип для создания
  **Плановой заявки**. Подробнее читайте в статьях: [Типы заявок](https://wiki.hubex.ru/docs/FAQ/RU/admin/TicketType.html.txt), [Стадии заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/StageType.html.txt), [Жизненный цикл заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/TicketLifeCycle.html.txt).

После заполнения всех полей нажмите кнопку **Сохранить и запланировать**. Таким образом вы запустите
процесс автоматического создания **Заявок**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/PlannedTicket.jpg)

##### Просмотр и редактирование Плановой заявки

Для редактирования **Плановой заявки** необходимо сначала приостановить ее действие. Для этого зайдите
в **Плановую заявку** и нажмите кнопку **Приостановить**. Теперь вы можете вносить
изменения. После внесения изменений необходимо заново нажать кнопку **Сохранить и запланировать**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PlannedTickets/Pause.jpg)

##### Плановые заявки в календаре

В web-приложении для календаря на месяц доступно отображение плановых заявок. Подробнее читайте в статье: [Календарь заявок](https://wiki.hubex.ru/docs/FAQ/RU/user/Calendar.html.txt).

Плановые заявки будут располагаться как в самом календаре на запланированную дату, так и на отдельной вкладке
**Планировщик**.

Например, если у вас запланирована **Заявка** на 19 апреля, то вы увидите ее в календаре отметку
**Планировщик 1** на дату 19 апреля, где 1 - это количество плановых заявок.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Calendar/PlanTicket.jpg)

Обратите внимание! Во вкладке **Планировщик** отображаются все плановые заявки на выбранный месяц. Если
вы выберете конкретный день, в который запланирована, например, одна плановая заявка, то на вкладке **Планировщик**
отобразится именно одна эта заявка. Чтобы вернуть отображение всех плановых заявок на выбранный месяц, еще раз
щелкните правой кнопкой мыши по выбранному дню, чтобы снять фокус с этого дня.

Если в настройках плановой заявки не указан исполнитель, то по кнопке **+** на вкладке **Планировщик**
вы можете выбрать подходящего сотрудника и назначить его на выполнение плановой заявки.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Calendar/ChoiceUser.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Calendar/ChoiceUser2.jpg)
