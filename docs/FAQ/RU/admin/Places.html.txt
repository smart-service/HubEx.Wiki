## Что такое участок в HubEx

**Участок** – это классификация **Объектов**, **Типов заявок**, **Сотрудников**, **Заказчиков** по различным критериям: географическим, функциональным и т.д.

С помощью настройки **Ролей** участками можно разграничить доступ **Сотрудников** и **Заказчиков** к работе с определенными **Объектами**, **Типами заявок** и **Заявками**. Подробнее читайте в статье: [Настройка ролей: Как настроить доступ пользователей в разрезе участков](https://wiki.hubex.ru/docs/FAQ/RU/admin/Roles.html.txt#exrole).

Например, можно создать участок «Электрооборудование» и отнести к нему все подходящие **Объекты**. Далее создать **Тип заявки**, например, «Ремонт электрооборудования», который будет иметь специфический жизненный цикл, и указать у него участок «Электрооборудование». Затем указать у **Сотрудника** тот же участок. Таким образом, этот **Сотрудник** сможет создавать **Заявки** с типом «Ремонт электрооборудования» и выбирать **Объекты** только по участку «Электрооборудования».

## Как создать участок

Для создания и настройки нового **Участка** перейдите в "Консоль администратора", далее в левом боковом меню выберите раздел **"Структура предприятия" - подраздел "Участки"**. Нажмите на кнопку **"Создать участок"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Places.jpg)

Поля в окне создания нового участка:

* **Название участка**: обязательное поле для создания участка, заполняется в свободной форме. Например, его название может быть "г. Калининград", "Северо-западный ФО", "Московский район" и т. п.;
* **Описание участка**: необязательное поле для создания участка, заполняется в свободной форме;
* **Идентификатор из внешней системы**: заполняется в том случае, если вы хотите связать HubEx с внешней системой, например 1С;
* **Назначить участком по умолчанию**: если установить галочку в этом параметре, то создаваемый или редактируемый вами участок будет по-умолчанию присваиваться новым создаваемым **Объектам**, **Сотрудникам** и **Заказчикам**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/NewPlace.jpg)

У каждого **Участка** есть статистика по количеству связей с **Объектами**, **Пользователями** (**Сотрудники + Заказчики**) и **Типами заявок**. Статистика необходима для принятия решений об удалении **Участка**, переноса всех элементов, относящихся к **Участку** на другой и т.д.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Statistic.jpg)

После создания нового **Участка** необходимо указать его в интересующих вас карточках **Объектов**, **Сотрудников**, **Заказчиков** и **Типов заявок**.

## Участок в объектах/оборудовании

Выбор участков в карточке объекта находится на вкладке **Объект|Оборудование** внизу страницы (левое боковое меню HubEx раздел **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"**).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Object.jpg)

**Обратите внимание!** Для объекта/оборудования можно указать несколько участков.

**Обратите внимание!** При создании нового участка во всех уже созданных в системе объектах/оборудовании новый участок нужно будет указать **вручную**, автоматически он не добавится.

## Участок в сотрудниках

Для сотрудника добавить участок можно на вкладке **"Квалификация"** (левое боковое меню HubEx, раздел **"Пользователи"** - подраздел **"Сотрудники"**).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Employee.jpg)

**Обратите внимание!** Для cотрудника можно указать несколько участков.

**Обратите внимание!** При создании нового участка во всех уже созданных в системе сотрудниках новый участок нужно будет указать **вручную**, автоматически он не добавится.

## Участок в заказчиках

Участок в карточке заказчика можно указать на вкладке **"Квалификация"** (левое боковое меню HubEx, раздел **"Пользователи"** - подраздел **"Заказчики"**).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Customer.jpg)

**Обратите внимание!** Для заказчика можно указать несколько участков.

**Обратите внимание!** При создании нового участка во всех уже созданных в системе заказчиках новый участок нужно будет указать **вручную**, автоматически он не добавится.

## Участок в типах заявки

**Тип заявки** - это справочник, который находится в "Консоли администратора" в левом бокового меню, раздел **"Настройки заявки"** - подраздел **"Типы заявок"**.

Если вам необходимо, чтобы определенный тип заявки со своим жизненным циклом использовался на определенном участке, то укажите этот участок в карточке типа заявки

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Places/Type.jpg)

**Обратите внимание!** Для типа заявки можно указать несколько участков.

**Обратите внимание!** После создания нового участка, новый участок **автоматически** добавится во все ранее созданные **Типы заявок**.
