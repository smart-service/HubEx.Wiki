#### Быстрые фильтры и массовые действия для списка объектов

В этом разделе вы узнаете:

* [Как сохранять и использовать быстрые фильтры](https://wiki.hubex.ru/docs/FAQ/RU/user/GroupActions.html.txt#filters);
* [Как выбрать несколько объектов и изменить их параметры](https://wiki.hubex.ru/docs/FAQ/RU/user/GroupActions.html.txt#group).

##### Быстрые фильтры для списка объектов

Для быстрого поиска по **Объектам** воспользуйтесь панелью **Фильтры**. Она находится
справа в меню **Объекты|Оборудование - Объекты|Оборудование**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/FilterObj.jpg)

С помощью фильтров вы можете установить отборы по различным полям **Объекта**.

Например, вам необходимо найти все **Объекты** определенного заказчика. Тогда на панели фильтров в
поле **Компания-владелец** нажмите кнопку **+Добавить** и
выберите вашего заказчика из списка. Затем нажмите кнопку **Выбрать**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Customer.jpg)

Список объектов автоматически отфильтруется по выбранному заказчику.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Customer2.jpg)

Для отмены фильтрации нажмите на кнопку сброса фильтров.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Customer3.jpg)

Вы можете сохранять настройки фильтрации. Это сэкономит ваше время на поиск необходимых **Объектов**.
Например, сохраним фильтрацию по **Компании-владельцу** и **Участку**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Filters.jpg)

Введите название быстрого фильтра.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/FiltersName.jpg)

Быстрый фильтр отобразится над списком всех **Объектов**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/FastFilters.jpg)

По кнопке **Все быстрые фильтры** откроется дополнительная панель со списком всех быстрых фильтров.
Здесь вы можете удалять неиспользуемые сохраненные фильтры.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/FilterDel.jpg)

##### Массовые действия для списка объектов

С помощью массовых действий над объектами можно быстро изменить/добавить/удалить определенные параметры сразу в
нескольких карточках **Объектов**:

* заполнить/заменить Родительский объект;
* добавить Вид работ;
* удалить Вид работ;
* добавить Участок;
* удалить Участок;
* установить/заменить Ответственного за объект;
* заполнить/заменить Тип объекта;
* заполнить/заменить Класс объекта;
* заполнить/заменить Компанию-владельца;
* установить/отключить Мобильность объекта;
* снять/установить публикацию объекта;
* добавить Навык;
* удалить Навык;
* установить/заменить график работы;
* добавить Чек-лист;
* удалить Чек-лист.

##### Как выбрать несколько объектов и изменить их параметры

Откройте список всех **Объектов** в меню **Объекты|Оборудование - Объекты|Оборудование**:

1. С помощью фильтров сделайте выбор группы объектов, для которых необходимо внести изменения. Например, необходимо
   изменить **Компанию-владельца** у объектов, принадлежащих **Кофейне Новый день**.
   Поэтому с помощью фильтров отбираем
   все объекты с **Коммпание-владельцем = Кофейня Новый день**. Таким образом получим отфильтрованный
   список объектов.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Objects.jpg)

   Использование фильтров целесообразно при большом количестве объектов в списке для того, чтобы минимизировать
   ошибку выбора объектов вручную. Данный шаг не является обязательным.
2. Теперь нужно включить режим массового выбора объектов и их редактирования. Наведите курсором мышки слева от
   любого объекта из отфильтрованного списка. Вы увидите, как появится окошко для установки флажка выбора.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/SelectEmpty.jpg)

   Установите флажок. Сверху появится панель редактирования. Выбрать все объекты на странице можно с помощью
   установки флажка в панели редактирования.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/SelectAll.jpg)

   Вы можете снять флажки с тех объектов, которые не должны измениться.
3. Нажмите кнопку **Редактировать**. Откроется форма для выбора параметров. Чтобы добавить несколько
   параметров нажмите
   **+**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/Edit.jpg)
4. Выберите параметры, которые хотите изменить. Например, выбираем **Компанию-владельца = Кофейня Доброе
   утро**. А также удалим ненужные виды
   работ. Для этого выбираем пункт **Вид работ (Удалить)** и флажками отмечаем те пункты, которые
   хотим убрать из карточек.

   Обратите внимание! Список **Видов работ** и **Участков** выводится полностью
   (выводятся все значения введенные в справочник **Виды работ** и **Участки**).

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/ParamsSelect.jpg)
5. Нажмите кнопку **Применить**.

При работе с отфильтрованным по старому параметру **Компании-владельца** списком объектов после
изменения владельца список объектов станет пуст.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/ListEmpty.jpg)

Проверим, правильно ли изменился владелец. Сделаем новый отбор в фильтрах по **Компании-владельцу = Кофейня
Доброе утро**. Теперь все обработанные ранее **Объекты** принадлежат новой компании.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActions/NewFilter.jpg)

Обратите внимание! Для смены **Типа объекта** через групповое редактирование есть нюанс: заменить Тип
объекта (Тип
оборудования) можно только для дочерних объектов.

В списке **Объектов** сделайте отбор с помощью фильтров или вручную тех **Объектов** (как
описано выше в текущей статье), для
которых необходимо изменить **Тип оборудования** (**Тип объекта**).

В примере мы отобрали **Объекты**, в наименовании которых есть слово "кофемашина", а также те, у которых
**Тип оборудования
= Объект**, чтобы заменить его на тип **Оборудование**.

Затем нажмите кнопку **Редактировать**, чтобы открыть форму для изменения параметров.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/MassEditting.jpg)

В боковой форме укажите, какой вид параметра нужно изменить. В нашем примере меняем **Тип оборудования**
(**Тип объекта**) на значение **Оборудование**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/ChangeType.jpg)
