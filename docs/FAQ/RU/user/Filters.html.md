##### Работа со списком Заявок

В списке заявок предусмотрено два режима представления всех **Заявок**:

1. **Карточное отображение заявок**: представляет список **Заявок** в виде отдельных
   карточек;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/CardList.jpg)

2. **Табличное отображение заявок**: представляет список **Заявок** в виде таблицы. Для
   табличного отображения вы можете
   самостоятельно настроить поля, которые будут отображаться в таблице. Для этого нажмите значок шестеренки и
   выберите необходимые поля.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/TableList.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/TableList2.jpg)

Также в табличное представление **Заявок** можно добавить отображение дополнительных полей.
Дополнительные поля могут быть настроены для формы **Заявки** в консоли администратора. Подробнее
читайте в статье: [Создание дополнительных
полей](https://wiki.hubex.ru/docs/FAQ/RU/user/AdditionalFields.html.md).

Это позволит сразу видеть всю необходимую информацию в таблице, не проваливаясь внутрь **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/AddField.jpg)

В табличном отображении **Заявок** доступно для отображения поле **Дата изменения** - это
дата последних внесенных в **Заявку** изменений.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Date.jpg)

Все изменения по **Заявке** хранятся на вкладке **История изменений** в
**Заявке**. В поле **Дата изменений** отображается последняя строка из истории.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/HistoryOfChange.jpg)

Также в табличном отображении списка вы можете настраивать очередность полей. Для этого зажмите и удерживайте правую
кнопку мышки на наименовании колонки и перенесите ее в нужное место.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/List.jpg)

Обратите внимание! Список заявок для каждого пользователя может отличаться. Это связано с настройкой
**Ролей**. Для
разных **Ролей** пользователей могут быть настроены ограничения, например по работе с
**Участками**. То есть пользователь
будет видеть только **Заявки**, находящиеся на его участке. Подробнее читайте в статье: [Настройка ролей: как настроить доступ
пользователей в разрезе участков](https://wiki.hubex.ru/docs/FAQ/RU/admin/Roles.html.md#exrole).

##### Использование фильтров

Для быстрого поиска по **Заявкам** воспользуйтесь кнопкой **Фильтры**. Она находится в правом верхнем углу в меню **Все заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FiltersButton.jpg)

С помощью фильтров вы можете установить отборы по различным полям **Заявки**. Фильтрация **Заявок** возможна
также по дополнительным полям, они отображаются в конце списка всех фильтров. Подробнее читайте в статье: [Создание дополнительных полей для
заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/AdditionalFields.html.md).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Filters.jpg)

Фильровать можно по списочным форматам, текстовым, форматам дат и времени, числовым форматам полей. Для числовых форматов дополнительных полей
доступен выбор настроек (**Равно**, **Диапазон**, **Больше чем** и**Меньше чем**). Для списочных форматов полей доступен соответствующий выбор значений - единичный или множественный выбор. Таким образом вы сможете быстро и эффективно находить **Заявки**. После
указания параметров необходимо нажимать кнопку **Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Filter2.jpg)

Например, вам необходимо найти все **Заявки** по определенному оборудованию. Тогда в окошке фильтров в
поле **Объекты**
выберите оборудование и нажмите на кнопку **Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FilterObj.jpg)

При использовании дополнительных полей формата дат и списков можно отфильтровать **Заявки**, например, по дате осмотра объекта, инструментам, типу выезда (первичный/вторичный) или другим параметрам, для которых используются форматы списков и дат.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FilterList.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/ListsFilter.jpg)

Для отмены фильтрации нажмите на кнопку сброса фильтров.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/ClearFilters.jpg)

##### Настройка панели фильтрации

По клику на кноку **Фильтры**открывается панель фильтров с параметрами фильтрации по **Заявкам**. При наведении курсора на область поля параметра появляется значок перемещения.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterSettings2.png)

Перетаскиванием поля вверх или вниз вы можете изменять порядок отображения фильтров. В процессе перетаскивания вы увидите, как другие поля автоматически подстраиваются, чтобы освободить место. Все изменения расположения полей будут сохранены автоматически.

Добавить необходимые поля для фильтрации можно по клику на кнопку **Добавить**, расположенную в нижней части панели фильтров.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterSettings3.png)

Откроется окно выбора полей, в верхней части которого расположены основные поля, в нижней - дополнительные поля, созданные в вашем тенанте.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterSettings4.png)

Выбранные вами поля на этом шаге будут отображаться на панели фильтров в той последовательности, в которой вы их добавляли.

Если вы удалите поле с панели фильтрации, то снова добавить его можно также через кнопку **Добавить**.

##### Сохранение и использование быстрых фильтров

Вы можете сохранять настройки фильтрации. Это сэкономит ваше время поиска необходимых **Заявок**.
Например, вы часто
работаете с **Заявками** определенного **Заказчика** или с **Заявками** по
конкретному **Объекту**. Настройте необходимые
пункты и нажмите на кнопку сохранения фильтра.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/SavingFilters.jpg)

Введите название быстрого фильтра.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FiltersName.jpg)

Быстрый фильтр отобразится над списком всех **Заявок**. Чтобы просмотреть все доступные быстрые фильтры
и удалить неактуальные, нажмите на кнопку **Все быстрые фильтры**. В списке фильтров по нажатию на
крестик можно удалять неиспользуемые быстрые фильтры. Быстрые фильтры - это индивидуальные настройки для конкретного
пользователя. Каждый пользователь может настраивать и сохранять только необходимые ему фильтры.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FastFilters.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/FastFilters2.jpg)

##### Фильтрация закрытых Заявок

**Заявка** в стадии **Закрыта** не отображается в списке всех заявок, так как в системе по
умолчанию настроен фильтр
показывать только **Открытые заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/ActiveFilters.jpg)

Для того чтобы увидеть все **Заявки**, нажмите кнопку **Фильтры**. Выберите в пункте
**Состояние заявки** один из вариантов:

* **Только закрытые**: вы увидите в списке только закрытые **Заявки**;
* **Все заявки**: в списке отразятся и открытые, и закрытые **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/ClosedTickets.jpg)

##### Фильтрация удаленных Заявок

Вы можете удалить **Заявку**, созданную по ошибке. Подробнее читайте в статье: [Переход заявки по стадиям и
удаление заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/ChangingStatus.html.md#deletetick). Она перестанет
отображаться в списке всех **Заявок**.
Но с помощью фильтра **Видимость заявок** ее можно будет найти. Выберите один из вариантов:

* **Только удаленные**: вы увидите в списке только удаленные **Заявки**;
* **Все заявки**: в списке отразятся и удаленные, и действующие **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/DeletedTickets.jpg)

Обратите внимание! Возможность просматривать удаленные заявки доступна только для учетной записи владельца
тенанта.

##### Поиск Заявок с помощью строки поиска

Помимо панели фильтров для поиска необходимых **Заявок** можно также использовать специальную строку
поиска. Строка
поиска находится в правом верхнем углу списка **Заявок**. При фильтрации по строке поиска над самой
строкой будет отображаться количество **Заявок**, подходящих под условия отбора.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/SearchString.jpg)

Поиск по строке доступен для списочного, табличного отображения, а также для режима отображения на карте.

Если часть слова, слово целиком, словосочетания или числа совпадут с данными в одной или нескольких
**Заявках**, то
такие **Заявки** останутся в списке, а неподходящие будут скрыты. Чем точнее будет поисковый запрос,
тем точнее будут результаты поиска. Поиск осуществляется одновременно по следующим полям:

* **Номер заявки**: для поиска по номеру можно вводить числа и буквы, если они используется в формате номера для более точного поиска.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Number.jpg)

  Обратите внимание! Текст из строки поиска будет основой для поиска по всем нижеперечисленным полям одновременно.
  Например, если вы напишите в строку поиска цифру 70 и она будет частью номеров **Заявок** и встречается
  в названии **Объектов**, то все
  такие **Заявки** будут отображаться в списке;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/SearchString2.jpg)

* **Описание заявки**: чем точнее и уникальнее будет задан поисковый запрос, тем лучше система обработает поиск по
  описанию заявки. Например, используйте для поиска уникальные фразы, наименования материалов и т.д.;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Text.jpg)

* **Объект**: если часть слова, слово целиком, словосочетания или числа совпадут с наименованием
  **Объекта**,
  указанного в одной или нескольких **Заявках**, то такие **Заявки** останутся в списке,
  а неподходящие будут скрыты;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Object.jpg)

* **Адрес объекта**: адрес для поиска можно вводить, как часть названия города, улицы, сочетания улицы и
  номера дома через запятую.

  Обратите внимание! Чем точнее вы введете данные адреса, тем точнее будет отработан поиск подходящих **Заявок**.
  Например, улицу и номер дома через запятую. Но не забывайте, что поиск будет осуществлен одновременно по
  всем полям, то есть если заданный в строке поиска адрес будет фигурировать в описании к Заявке по другому
  адресу, то она все равно отобразится в списке подходящих.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/Adress.jpg)

* **Вид работ**: если часть слова, слово целиком, словосочетания совпадут с наименованием
  **Вида работ**,
  указанного в одной или нескольких **Заявках**, то такие **Заявки** останутся в списке,
  а неподходящие будут скрыты.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Filters/WorkType.jpg)
