#### Создание атрибутов для чек-листа

Атрибут - это способ измерения/выражения для пункта **Чек-листа**. Атрибут используется в случае, когда требуется вписать
ответ в виде числа, даты, текстовой строки или прикрепить файл. Подробнее читайте в статье: [Чек-листы](https://wiki.hubex.ru/docs/FAQ/RU/user/Checklists.html.md).

Ознакомьтесь с обучающим видеороликом **Как создать дополнительные поля в Hubex?** или прочтите пошаговую инструкцию ниже.

Создать новый атрибут можно в [консоли
администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.md) в меню **Настройки системы - Дополнительные поля** по кнопке **Создать
поле**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/TicketAttribute/AddAtt.jpg)

Заполните параметры нового атрибута:

* **Наименование поля** в свободной форме;
* **Тип поля** устанавливается исходя из типа данных, которые планируется вносить с помощью данного атрибута.
  Например, если создавать атрибут для ввода площади гостиничного номера, то **Тип поля** будет равен значению **Дробное
  число**. Обратите внимание! В уже созданном атрибуте **Тип поля** изменить нельзя, для создания атрибута с другим типом необходимо будет создать новый атрибут;
* **Единица измерения** устанавливается исходя из заданного **Типа поля**. Площадь измеряется в метрах;
* В поле **Используется для** устанавливается значение **Чек-листы**;
* Параметр **Показывать в паспорте объекта** используется при создании дополнительных полей для **Объектов**.
