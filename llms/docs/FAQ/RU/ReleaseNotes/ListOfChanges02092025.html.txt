## 🚀 Системные справочники в дополнительных полях для Договоров и Компаний

**Раздел: Договоры / Компании / Администрирование > Дополнительные поля**

---

## 🌟 Что реализовано?

Мы расширили функциональность дополнительных полей для сущностей **Договоры** и **Компании**! Теперь вы можете использовать системные справочники HubEx напрямую в карточках договоров и компаний, обеспечивая единообразие данных и удобство работы.

---

## 🔧 Что нового?

### 📌 Поддержка системных справочников в дополнительных полях

При создании дополнительного поля в административной-панели для договоров и компаний теперь доступен тип:
**«Значение из справочника»** — выбор значения из одной из системных сущностей:

* Объекты/Оборудование
* Компании
* Сотрудники
* Заказчики
* Материалы

📌 Дополнительно можно включить **множественный выбор** через чек-бокс *«Разрешить выбирать несколько значений»*.

---

## 🎯 Использование в договорах и компаниях

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/39RKF2xK_image.png)

### Для Договоров:

* Добавляйте поля с привязкой к объектам, материалам или ответственным сотрудникам прямо в карточку договора.
* Например: «Объекты по договору», «Ответственные менеджеры», «Связанные материалы».

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/xqCSVDSf_image.png)

### Для Компаний:

* Добавляйте поля с привязкой к объектам, материалам или ответственным сотрудникам прямо в карточку компании.
* Например: «Дочерние компании», «Ключевые сотрудники», «Привязанные объекты».

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/zW0HtYXy_image.png)

---

## ⚠️ Важно

* Функциональность доступна для **Договоров** и **Компаний** (в дополнение к ранее реализованным **Заявкам**).

## 🚀 Управление материалами в HubEx

## 🌟 Что нового?

Мы рады представить новый функционал для удобного управления материалами в системе HubEx! Теперь начальники сервисных служб и другие ответственные сотрудники могут эффективно вести учет материалов для последующего списания инженерами.

## 🔧 Ключевые возможности

### 📋 Табличное представление материалов

* Полный список материалов (номеклатура) в удобном табличном формате
* Сортировка по наименованию (алфавитный порядок)
* Основные колонки: Наименование, Описание, Артикул, Ед. изм., Стоимость, ERP ID

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/Z161hQjk_image.png)

### 🔍 Поиск и фильтрация

* **Быстрый поиск** по названию материала и ERP ID
* **Фильтрация** по статусу удаления (скрыть/показать удаленные)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/4ZFJLT4r_image.png)

### ✏️ Редактирование материалов

* Боковая панель редактирования с полной информацией о материале
* Обновление основных параметров: Наименование, Описание, Артикул, Стоимость, ERP ID
* Работа с единицами измерения через системный справочник

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges02092025/KjTN4XmY_image.png)

### 🗑️ Управление статусом материалов

* **Удаление** отдельных материалов и массовое удаление
* **Восстановление** отдельных материалов и массовое восстановление удаленных материалов
* Визуальное обозначение удаленных материалов (бейдж с иконкой корзины)

## 🎯 Преимущества для пользователей

✅ **Централизованное управление** — все материалы в одном месте
✅ **Быстрый поиск** — мгновенный доступ к нужным позициям
✅ **Гибкое редактирование** — легкое обновление информации
✅ **Безопасное удаление** — восстановление случайно удаленных материалов
✅ **Массовые операции** — экономия времени при работе с большими списками

## 📊 Как начать работать?

1. Перейдите в раздел "Склады | Материалы" > "Материалы"
2. Используйте поиск для быстрого нахождения нужных позиций
3. Редактируйте материалы через боковую панель
4. Управляйте статусом материалов с помощью удаления/восстановления

## ⚠️ Важная информация

* **Функционал ручного создания новых материалов** будет добавлен в следующих обновлениях
* Импорт материалов осуществляется через существующий функционал Инвентаризации (в разделе "Склады | Материалы" > "Остатки на складе")
