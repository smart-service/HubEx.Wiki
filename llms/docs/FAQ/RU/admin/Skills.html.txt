## Что такое навык в HubEx

Навыки – это дополнительные, особые умения, необходимые для выполнения определенной заявки.

## Для чего нужны навыки в HubEx

Навыки позволяют дополнительно классифицировать **Объекты** и **Сотрудников** (исполнителей).

С помощью навыков можно детализировать особые знания, которыми должен обладать специалист для работы с объектом обслуживания. Например, знать английский язык для настройки приборов на английском языке.

Навыки используются для настройки [правил автоназначения исполнителя](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/RulesOfChoice.html.txt). Укажите навыки в карточке **Объекта** (вкладка **"Обслуживание"**) и карточке **Сотрудника** (вкладка **"Квалификация"**), тогда с помощью правил автоназначения исполнителей HubEx будет автоматически подбирать исполнителей на выполнение **Заявки** по навыкам. Подробнее читайте в статьях: [Правила автоназначения исполнителя](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/RulesOfChoice.html.txt).

## Как создать навык

Создать **Навык** можно в [консоли администратора](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.txt) в разделе левого бокового меню **"Структура предприятия"** - подраздел **"Навыки"** по кнопке **"Создать навык"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/Skills/Skill.jpg)

Рассмотрим заполнение карточки **Навыка**:

* **Название навыка**: обязательное поле для заполнения, заполняется в свободной форме;
* **Описание**: необязательное поле для заполнения, вы можете внести подробности о содержании навыка;
* **Необязательный**: дополнительная классификация навыков, которая используется в настройке правил автоназначения исполнителей на заявки.

  Правила выбора могут быть настроены с учетом обязательных навыков, с учетом необязательных навыков и с учетом обеих групп навыков.

## Навыки в сотрудниках

Для сотрудника добавить навык можно на вкладке **"Квалификация"** (левое боковое меню HubEx, раздел **"Пользователи"** - подраздел **"Сотрудники"**).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/SkillsNew/SkillsNew1.png)

**Обратите внимание!** Для cотрудника можно указать несколько навыков.

**Обратите внимание!** При создании нового навыка, новый навык нужно будет указать для интересующего вас сотрудника **вручную**, автоматически в карточку сотрудника новый навык не добавится.

## Навыки в объектах

Выбор навыков в карточке объекта находится на вкладке **"Обслуживание"** (левое боковое меню HubEx раздел **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"**).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/SkillsNew/SkillsNew2.png)

**Обратите внимание!** Для объекта/оборудования можно указать несколько навыков.

**Обратите внимание!** После создании нового навыка, в интересующих вас объектах/оборудовании, новый навык нужно будет указать **вручную**, автоматически в карточку объекта/оборудования новый навык не добавится.
