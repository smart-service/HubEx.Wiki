<p>Мобильное приложение для заказчика позволяет оперативно создавать <strong>Заявки</strong>, подписывать акты
    выполненных работ и
    ставить оценку работы исполнителей.</p>
<p>Данная статья будет полезна как для заказчика, так и для администрирующего персонала со стороны исполнителя.</p>

<p>Заказчик – это пользователь системы, который может создавать заявки и следить за ходом их выполнения. Заказчиком
    может выступать, например, менеджер компании, с которой компания-исполнитель заключила договор на предоставление услуг
    обслуживания, доставки и т.д.
</p>

<p>Скачать мобильное приложение <strong>HubEx заказчик</strong> можно по ссылкам <a
        href="https://play.google.com/store/apps/details?id=ru.hubex.customer">для Android</a> или <a
        href="https://apps.apple.com/ru/app/hubex-%D0%B4%D0%BB%D1%8F-%D0%B7%D0%B0%D0%BA%D0%B0%D0%B7%D1%87%D0%B8%D0%BA%D0%B0/id1386631658">для
    iOS</a>. </p>

<h5 id="ticketmob">Создание заявки в мобильном приложении HubEx заказчик</h5>

<p>Мобильное приложение для заказчика позволяет оперативно создавать заявки, подписывать акты выполненных работ и
    ставить оценку работы исполнителей.</p>
<p>Внизу располагаются кнопки меню приложения:</p>
<ul>
    <li><strong>Мои</strong>: это <strong>Заявки</strong>, созданные от имени текущего пользователя;</li>
    <li><strong>Согласование</strong>: здесь находятся <strong>Заявки</strong>, которые вам нужно согласовать;</li>
    <li><strong>Все</strong>: на текущей вкладке находятся все <strong>Заявки</strong> только по вашей компании и вашим <strong>Объектам</strong> (в том числе здесь
        отображаются <strong>Заявки</strong>, которые были созданы вашими коллегами по вашей компании и вашим <strong>Объектам</strong>);
    </li>
    <li><strong>Избранное</strong>: вкладка с <strong>Заявками</strong>, добавленными в «избранное»;</li>
    <li><strong>Архив</strong>: здесь находятся закрытые <strong>Заявки</strong>.</li>
</ul>


</p>
<p>У каждой <strong>Заявки</strong> в списке отображается ее статус, на каком этапе находится заявка – В работе, Выполнена и т.д.</p>

<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Ticket3.jpg"/>
</div>


<p>Для создания новой <strong>Заявки</strong> нажмите кнопку подачи заявки в правом нижнем углу.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Ticket2.jpg"/>
</div>
<p>В мобильном приложении <strong>HubEx заказчик</strong> создать <strong>Заявку</strong> можно несколькими способами. Ниже будут представлены самые
    основные из них. А в статье
    <a href="https://wiki.hubex.ru/docs/FAQ/RU/user/AlternativeWays.html">Альтернативные способы подачи заявки</a> можно
    ознакомиться с дополнительными вариантами подачи <strong>Заявки</strong> через сайт,
    электронную почту и электронный паспорт объекта.</p>
<p>Базовые варианты подачи заявки для пользователя мобильного приложения:</p>
<ol>
    <li>Первый способ заключается в выборе <strong>Объекта</strong> или <strong>Оборудования</strong> из списка: в открывшемся окошке необходимо выбрать
        <strong>Объект</strong> для подачи <strong>Заявки</strong>.
        <p>После выбора <strong>Объекта</strong> откроется форма <strong>Заявки</strong>.</p>
        <p>Поля, отмеченные красной звездочкой, являются обязательными для заполнения. Например, необходимо заполнить
            <strong>Тип заявки</strong>: Заявка на ремонт, Гарантийный ремонт, Задача и т.д.
            В поле <strong>Описание заявки</strong> введите подробную информацию о том, что случилось на объекте и какие работы
            необходимо выполнить.</p>
        <p>После заполнения формы нажмите кнопку <strong>Создать заявку</strong>.</p>

        <p>
        <div>
            <img style="margin: 0 auto; display: block; max-width: 70%;"
                 src="/attachments/images/FAQ/USER/CustomerApp/Ticket.jpg"/>
        </div>
        </p>
    </li>
    <li>Второй способ заключается в подаче <strong>Заявки</strong> с помощью сканирования QR-кода: в открывшемся окне нажмите кнопку
        подачи заявки по QR-коду в правом нижнем углу.
        <p>В приложении запустится сканер QR-кодов, который необходимо навести на QR-код, которым промаркирован
            объект.</p>
        <p>
        <div>
            <img style="margin: 0 auto; display: block; max-width: 70%;"
                 src="/attachments/images/FAQ/USER/CustomerApp/QR2.jpg"/>
        </div>
        </p>

        <p>После считывания QR-кода откроется форма <strong>Заявки</strong>, где необходимо выбрать <strong>Тип заявки</strong>: заявка на ремонт,
            гарантийный ремонт, задача и т.д.</p>
        <p>В поле <strong>Описание заявки</strong> введите подробную информацию о том, что случилось на объекте и какие работы необходимо
            выполнить.</p>
        <p>После заполнения формы нужно нажать кнопку <strong>Создать заявку</strong>.</p>
        <p>Созданная <strong>Заявка</strong> состоит из нескольких вкладок, которые располагаются внизу. Вкладки <strong>Чек-листы</strong> и <strong>Выполнение</strong>
            заполняются исполнителем по заявке.</p>
        <p>На вкладке <strong>Акт работ</strong> после выполнения <strong>Заявки</strong> формируется акт.</p>
        <p>
        <div>
            <img style="margin: 0 auto; display: block; max-width: 70%;"
                 src="/attachments/images/FAQ/USER/CustomerApp/QR.jpg"/>
        </div>
        </p>
    </li>
</ol>


<p>Для каждой вкладки с <strong>Заявками</strong> доступен поиск и фильтрация <strong>Заявок</strong> с помощью инструментов в правом верхнем углу.</p>

<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Filter.jpg"/>
</div>


<h5 id="mapmob">Просмотр карты в заявке</h5>
<p>В уже созданной <strong>Заявке</strong> вы можете посмотреть на карте, где находится исполнитель, назначенный на эту <strong>Заявку</strong>. </p>
<p>Откройте <strong>Заявку</strong> и ниже поля <strong>Адрес</strong> нажмите на карту. Карта развернется на весь экран: исполнитель и заявка будут
    отображаться разными метками.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Map.jpg"/>
</div>

<h5 id="chatmob">Переписка в чате по заявке</h5>
<p>По каждой <strong>Заявке</strong> можно вести отдельную переписку в разделе <strong>Сообщения</strong>. Таким образом, вы оперативно можете узнать у
    исполнителя, на каком этапе выполнения находится <strong>Заявка</strong> и задать любые вопросы. </p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Chat2.jpg"/>
</div>

<p>Чат в мобильном приложении доступен в самой <strong>Заявке</strong> в правом верхнем углу.
    Также все сообщения по всем <strong>Заявкам</strong> отображаются в меню <strong>Сообщения</strong>. Когда приходит новое сообщение по какой-либо
    <strong>Заявке</strong>, появляется значок с уведомлением о непрочитанном сообщении и приходит push-уведомление. </p>

<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Chat.jpg"/>
</div>

<h5 id="cancelticket">Отмена заявки</h5>

<p>В зависимости от настроек HubEx <strong>Заявку</strong> можно переводить на различные этапы. Например, вы можете отказаться от
    <strong>Заявки</strong>, принять работы по <strong>Заявке</strong> или вернуть ее на доработку и т.д.</p>
<p>Например, после создания <strong>Заявки</strong>, до выезда исполнителя на объект, вы можете отказаться от <strong>Заявки</strong>, если поломка была
    устранена собственными силами или актуальность <strong>Заявки</strong> пропала.</p>
<p>Для этого откройте заявку и внизу нажмите кнопку <strong>Отменить заявку</strong>.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Cancel.jpg"/>
</div>


<h5 id="actmob">Приемка выполненных работ и подпись акта</h5>
<p>Исполнитель результаты работ по заявке вносит на вкладке <strong>Выполнение</strong>. Выполненные работы формируют Акт выполненных
    работ на соответствующей вкладке.</p>
<p>Принять выполненные работы и подписать акт можно только в мобильном приложении заказчика <strong>HubEx заказчик</strong>.</p>
<p>Принять работу и подписать акт можно только, когда исполнитель внес выполненные работы.</p>
<p>Тогда в мобильном приложении заказчика в заявке в нижней части появляются две кнопки Принять работы и Не принимать
    работы.</p>
<p>Если работы выполнены в соответствии с требованиями, то их необходимо принять и перейти к подписанию акта. Если
    заказчик не удовлетворен выполнением заявки, то работы можно не принимать и вернуть заявку на доработку исполнителю
    по нажатию кнопки Не принимать работы.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Act3.jpg"/>
</div>

<p>Акт доступен на вкладке <strong>Акт работ</strong>. Подписать акт можно по нажатию на область <strong>Подпись</strong> или кнопку <strong>Получить
    подпись</strong>. Окно для подписи откроется в альбомной ориентации: необходимо повернуть телефон влево и пальцем сделать
    подпись на экране. После чего нужно нажать значок галочки в верхнем правом углу. Если подпись вас устраивает, то
    далее нажмите кнопку <strong>Применить</strong>.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Act2.jpg"/>
</div>
<p>Подпись появится в акте. Теперь его можно будет распечатать из клиентского портала HubEx.</p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 70%;"
         src="/attachments/images/FAQ/USER/CustomerApp/Act.jpg"/>
</div>

<h5 id="ratingmob">Оценка работы исполнителя</h5>
<p>Для выполненной <strong>Заявки</strong> доступна возможность поставить оценку исполнителя. Для этого откройте <strong>Заявку</strong> и вверху нажмите
    на поле <strong>Оцените качество исполнения заявки</strong>.</p>
<p>Перед вами откроется окошко, где можно с помощью звездочек установить оценку качества выполненных работ и
    оперативность. Также можно дополнить оценку текстовым комментарием. </p>
<p>После установки оценок нажмите кнопку <strong>Оценить</strong>.</p>

    <img style="margin: 0 auto; display: block; max-width: 70%;"
