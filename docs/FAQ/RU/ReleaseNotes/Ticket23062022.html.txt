#### Упрощенная форма заявки: обновление от 23.06.2022

Оставьте только необходимые поля на форме **заявки**: теперь поля **Объект**, **Заказчик** и многие другие не являются обязательными для заполнения.

Вы можете подавать **Заявку** без привязки к конкретному **Объекту** или **Заказчику**, можете не указывать **Вид работ** и адрес.

Это актуально для работы с внутренними задачами организации: с текущим обновлением вы можете создавать заявки для бухгалтеров, it-специалистов, курьеров и т.д.

Теперь использовать HubEx можно не только для выполнения заявок по обслуживанию клиентов, но и в целях своей организации.

##### Настройка упрощенной формы заявки

Отключить лишние поля для формы заявки можно в консоли администрирования в меню **Настройки заявки - Поля на форме заявки**.

Выберите **Роли** и **Стадии заявки** и для каждого из необязательных полей оставьте настройку чтения и заполнения пустой.

Подробнее о настройке полей можно прочитать в статье нашей базы знаний [Поля на форме заявки: настройка доступа к полям заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/ElementsOfInterface.html.txt).

Например, можно убрать видимость **Объекта**, **Заказчика**, **Вида работ**, **Контактов по объекту и т.д.**

Обязательными полями для формы остались только **Тип заявки** и **Критичность**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Role.png)

После отключения необязательных полей форма заявки будет выглядеть следующим образом.
