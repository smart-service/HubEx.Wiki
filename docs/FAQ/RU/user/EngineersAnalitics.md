<p>В HubEx представлены аналитические отчеты реализованные на платформе Power BI. С помощью бизнес-аналитики вы
    можете анализировать данные, получать ценные сведения для принятия бизнес-решений.</p>
<p>Каждый аналитический отчет реализован в виде панели с различными графиками, диаграммами, сводными
    таблицами и показателями.</p>
<p><Strong>Отчет по сотрудникам</Strong> в HubEx дает возможность проанализировать данные, собранные по
    <Strong>Сотрудникам</Strong> и <Strong>Заявкам</Strong>. С помощью этих визуализаций
    вы
    сможете
    ответить на
    вопросы - </p>

<p><Strong>Отчет по сотрудникам</Strong> находится в меню <Strong>Аналитика - Отчет по сотрудникам</Strong>. Данные
    в отчете обновляются один раз в
    сутки.
    Дата
    обновления отображается под наименованием отчета. </p>
<div>
    <img style="margin: 0 auto; display: block; max-width: 95%;"
         src="/attachments/images/FAQ/USER/EngineersAnalitics/XXXXXXXXXXXXXXXXXXXXXXXX.jpg"/>
</div>
<p>Представление панели отчетов основывается на данных в <Strong>Заявках</Strong> в разрезе <Strong>Сотрудников</Strong>.
    Сотрудники – пользователи, которые могут создавать и выполнять Заявки. Сотрудником может являться диспетчер вашей организации, который создает заявки в web-приложении, руководитель вашей
    организации, сервисный специалист (исполнитель), который работает по Заявкам. В контексте данного отчета будут рассматриваться данные именно по специалистам, которые хотя бы раз были назначены на Заявку.

    Подробнее читайте в статьях: <a
            href="https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingObjects.html#parentchild">Создание пользователя вашей
        системы
        объектов</a>, <a
            href="https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingTicket.html">Создание заявки</a>.
</p>
<p>В правом верхнем углу вы можете указать период отображения данных в отчете.</p>
<p>С помощью кнопки <Strong>Фильтров</Strong> можно выбрать <Strong>Тип заявки</Strong>,
    <Strong>Участок</Strong> и т.д. Данный фильтр
    будет применен ко всем диаграммам и показателям в панели отчетов.</p>
