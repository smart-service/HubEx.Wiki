#### Расширенный экспорт объектов: обновление от 17.01.2022

Два новых улучшения для списка **Объектов**:

* расширенный экспорт **Объектов**: теперь при экспорте **Объектов** можно выбирать поля для выгрузки;
* панель быстрых фильтров для списка **Объектов**: быстрые фильтры для отбора **Объектов** теперь на отдельной панели.

##### Расширенный экспорт Объектов

Теперь вы можете выбирать, какие данные по **Объектам** или **Оборудованию** необходимо выгрузить в табличный файл Excel.

Перейдите в меню **Объекты|Оборудование - Объекты|Оборудование** и нажмите кнопку **Экспорт объектов и оборудования**.

Выберите пункт - **Расширенный экспорт**. В списке укажите, какие поля требуется выгружать для **Объектов**. После выбора необходимых полей нажмите кнопку **Экспорт**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsExport17012022/1.png)

##### Быстрые фильтры для списка объектов

Наборы фильтров для списка **Объектов** теперь сохраняются на специальную панель быстрых фильтров.

Таким образом, вы получаете моментальный доступ к сортировке списка **Объектов** по различным параметрам, например, **Компании-владельцу**, **Виду работ**, **Участку** и т.д.

Перейдите в меню **Объекты|Оборудование - Объекты|Оборудование**. Для сохранения фильтрации в меню справа выберите необходимые параметры фильтрации. Например, если вам нужно работать со списком **Объектов** определенного заказчика - выберите нужное значение для параметра **Компания-владелец** и нажмите кнопку **Сохранить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsExport17012022/2.png)

Введите наименование для сохраняемого фильтра и нажмите кнопку **Сохранить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsExport17012022/3.png)

Сохраненные фильтры будут отображаться над списком **Объектов** в панели быстрых фильтров.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsExport17012022/4.png)

Чтобы удалить быстрый фильтр, нажмите на кнопку **Все быстрые фильтры** и в списке удалите необходимый.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ObjectsExport17012022/5.png)
