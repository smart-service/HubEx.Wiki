#### Как изменить тип существующего оборудования?

По умолчанию поле **Тип оборудования** в карточке **Объекта** после записи становится
неактивным, заблокированным для
редактирования. Чтобы изменить **Тип оборудования** в карточке **Объекта**, нажмите кнопку
**Снять с
публикации**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/ChangeOfObjectType.jpg)

Теперь **Тип оборудования** доступен для редактирования. После внесения изменений обязательно сохраните
карточку **Объекта**
и нажмите кнопку **Опубликовать**. Если **Объект** останется неопубликованным, вы не
сможете создавать по нему **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/ChangeOfObjectType2.jpg)

Также **Тип оборудования** можно поменять при массовом редактировании в меню **Объекты|Оборудование**.
При массовом
изменении **Объектов** снимать их с публикации не нужно.

Обратите внимание! При массовом редактировании **Объектов** сменить **Тип оборудования**
можно будет только для категории дочерних объектов.

В списке **Объектов** сделайте отбор с помощью фильтров или вручную тех **Объектов**, для
которых необходимо изменить **Тип оборудования** (**Тип объекта**).

В примере мы отобрали **Объекты**, в наименовании которых есть слово "кофемашина", а также те, у которых
**Тип оборудования
= Объект**, чтобы заменить его на тип **Оборудование**.

Затем нажмите кнопку **Редактировать**, чтобы открыть форму для изменения параметров.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/MassEditting.jpg)

В боковой форме укажите, какой вид параметра нужно изменить. В нашем примере меняем **Тип оборудования**
(**Тип объекта**) на значение **Оборудование**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangeOfObjectType/ChangeType.jpg)
