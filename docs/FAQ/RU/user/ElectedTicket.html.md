##### Копирование и удаление заявок

Копирование и удаление **Заявок** одни из базовых действий над **Заявками**. В случае, если вы создали **Заявку** и указали
некорректный **Тип заявки**, то вам нужно скопировать некорректную **Заявку**, изменить в новой **Заявке** тип, а некорректную
удалить.

Рассмотрим, как скопировать **Заявку**:

* копирование доступно только для списка заявок в карточном представлении. Поэтому сначала убедитесь, что включено
  карточное
  представление заявок;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/List.jpg)

* чтобы скопировать **Заявку**, нажмите на контекстное меню (три точки) и выберите пункт **Копировать**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Copy.jpg)

* перед вами откроется форма заявки, поля которой будут заполнены точно так же как и поля той **Заявки**, которую вы
  скопировали. Пустым останется поле **Номер заявки**, оно будет заполнено после сохранения или, если необходимо,
  установите номер вручную. Также при сохранении будет заполнено правило расчета крайнего срока закрытия, которое
  устанавливается в зависимости от выбранных в **Заявке** данных. Подробнее читайте в статье: [SLA - Правила расчета крайнего срока закрытия
  заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/SLA.html.md).

  В скопированной **Заявке**, до ее сохранения, вы сможете изменить **Тип заявки** и другие поля при
  необходимости. После внесения изменений нажмите кнопку **Сохранить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Ticket.jpg)

**Заявку** можно удалить на любой стадии. Например, если вы внесли неправильную информацию, создали
документ по
ошибке.

Удалить **Заявку** можно только в том случае, если у пользователя достаточно на это прав. Права
настраиваются с помощью
**Ролей** в консоли администратора. Подробнее читайте в статье: [Настройка ролей](https://wiki.hubex.ru/docs/FAQ/RU/admin/Roles.html.md).

Право удалять **Заявки** рекомендуется назначать только администраторам системы, начальникам служб. В
консоли администратора в меню **Администрирование - Роли** откройте
необходимую **Роль** и внесите изменения:

1. Перейдите во вкладку **Полномочия**, выберите блок **Заявки**. Напишите часть слова
   "удал" (от слова "удаление") в графе **Поиск**.
2. В списке из подобранных совпадений выберите пункты **Удаление заявок** и **Удалять незакрытые
   заявки**. Сохраните внесенные изменения.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ChangingStatus/Roles.jpg)

Удалить **Заявку** можно несколькими способами. Рассмотрим первый:

* включите карточное представление заявок;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/List.jpg)

* чтобы удалить **Заявку**, нажмите на контекстное меню (три точки) и выберите пункт **Удалить**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Delete.jpg)

* в диалоговом окне подтвердите удаление **Заявки**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Dialog.jpg)

* если вы подтвердили удаление **Заявки**, то она пропадет из общего списка. Но **Заявки** не удаляются из системы
  напрямую, а скрываются из списка. Об этом рассказываем дальше.

Второй способ удалить **Заявку**:

* откройте **Заявку** из списка (здесь уже не имеет значения формат отображения списка заявок - это может быть таблица или список);
* в правом верхнем углу нажмите контекстное меню (три точки) и выберите пункт **Удалить**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Ticket2.jpg)

* подтвердите удаление **Заявки**;

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Dialog2.jpg)

* если вы подтвердили удаление **Заявки**, то она пропадет из общего списка. Но **Заявки** не удаляются из системы
  напрямую, а скрываются из списка. Об этом рассказываем дальше.

И третий способ - это массовое удаление **Заявок**.

Откройте список всех **Заявок** в меню **Заявки - Все заявки**.

Обратите внимание! Массовые действия доступны для карточного и табличного представления списка. Подробно о форматах представления списка читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/docs/FAQ/RU/user/Filters.html.md).

Чтобы удалить несколько **Заявок**, необходимо:

1. С помощью фильтров сделать выбор группы **Заявок**, которые необходимо удалить. Например, необходимо
   удалить **Заявки** с видом работ **Гарантийный ремонт**. Поэтому с помощью фильтров отбираем
   все **Заявки** с **Видом работ = Гарантийный ремонт**. Таким образом получим отфильтрованный список.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Tickets2.jpg)

   Использование фильтров целесообразно при большом количестве **Заявок** в списке для того, чтобы минимизировать
   ошибку выбора их вручную. Данный шаг не является обязательным. Подробнее о работе с фильтрами читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/docs/FAQ/RU/user/Filters.html.md).
2. Теперь нужно включить режим массовых действий. Наведите курсором мышки слева от
   любой **Заявки** из отфильтрованного списка. Вы увидите, как появится окошко для установки флажка выбора.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectEmpty2.jpg)

   Установите флажок. Сверху появится панель редактирования. Выбрать все **Заявки** на странице можно с помощью
   установки флажка в панели редактирования. Либо продолжить выбор **Заявок** с помощью установки флажков вручную

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectedTick2.jpg)
3. Нажмите кнопку **Удалить**. Откроется диалоговое окно для подтверждения удаления.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Deleting.jpg)

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Deleting2.jpg)

Удаленные **Заявки** сразу исчезнут из списка.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/TicketList.jpg)

##### Где найти удаленные заявки

**Заявки** не удаляются из системы полностью. Удаленные **Заявки**
скрываются из общего списка. С
помощью фильтров вы можете отобразить **Все заявки** или показать **Только удаленные**.
Подробнее читайте в статье: [Почему нельзя
удалить объекты/сотрудников/заявки и т.д. навсегда?](https://wiki.hubex.ru/docs/FAQ/RU/user/DeletedObjects.html.md)

Откройте настройку фильтров и в разделе **Видимость заявок** выберите вариант **Только удаленные** и нажмите кнопку
**Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Filters.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Filters2.jpg)

В списке отобразятся только удаленные **Заявки**: у удаленных **Заявок** справа находится красный значок корзинки.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/DeletedTickets.jpg)

Обратите внимание! Возможность просматривать удаленные заявки доступна только для учетной записи владельца
тенанта.

##### Добавление заявки в избранное

Избранное - ваш персональный перечень **Заявок**. Данная функция предназначена для добавления важных **Заявок** в отдельный
список для быстрого доступа к ним.
Например, вы можете добавить в избранное **Заявки**, над которыми ведется продолжительная работа и так далее.

Чтобы добавить **Заявку** в избранное, откройте эту **Заявку** и нажмите контекстное меню в правом верхнем углу (три
точки) и выберите пункт **Добавить в избранное**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/ElectTicket.jpg)

Убрать из избранного **Заявку** можно также по нажатию на контекстное меню в правом верхнем углу и выберите пункт **Убрать
из избранного**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/ElectTicket2.jpg)

В общем списке избранные **Заявки** отображаются со значками золотого флажка.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/ElectedTickets.jpg)

Чтобы просмотреть отдельный список избранных **Заявок**, откройте настройку фильтров и в разделе **Избранные заявки**
выберите вариант **Только избранные заявки** и нажмите кнопку
**Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Filters3.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/Filters4.jpg)

Список заявок будет отфильтрован и вы увидите только избранные из них.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ElectedTicket/ElectedTickets2.jpg)
