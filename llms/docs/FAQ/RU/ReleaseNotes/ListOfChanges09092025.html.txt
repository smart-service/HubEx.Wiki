## 🚀 Автоматическая привязка видов работ к типам оборудования

### 🌟 Что нового?

Мы представляем масштабное обновление, которое значительно упрощает настройку и управление видами работ для оборудования! Теперь вы можете централизованно настраивать виды работ для типов оборудования, что автоматически применяется ко всем объектам данного типа при создании.

### 🔧 Ключевые возможности

#### 📋 Централизованное управление видами работ

* Привязка видов работ к типам оборудования через админку

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges09092025/dw3cKc13_image.png)

* Автоматическое применение настроек ко всем объектам выбранного типа при создании объекта

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges09092025/39YBJrVp_image.png)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges09092025/rGXycjrY_image.png)

* Гибкие опции обновления существующих объектов

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges09092025/4gKnyfq7_image.png)

#### 🛠️ Удобный интерфейс

* **Боковое окно** для редактирования типов оборудования
* **Множественный выбор** видов работ
* **Визуальное отображение** выбранных видов работ
* **Интеллектуальные подсказки** при изменении настроек

#### ⚡ Автоматизация процессов

* **Автозаполнение** видов работ при создании объектов
* **Умные уведомления** при изменении типа оборудования

### 🎯 Преимущества для пользователей

✅ **Экономия времени** - больше не нужно вручную настраивать виды работ для каждого объекта
✅ **Единообразие данных** - все объекты одного типа имеют одинаковые настройки
✅ **Уменьшение ошибок** - автоматическое применение правильных видов работ

### 📋 Как это работает?

#### Для администраторов:

1. Откройте настройки типа оборудования
2. Выберите нужные виды работ из списка
3. Сохраните изменения.

#### Для пользователей:

* При создании объекта автоматически подставляются виды работ из типа оборудования
* При изменении типа оборудования система предлагает обновить виды работ
* Возможность ручного переопределения при необходимости

## 🚀 Новые виджеты в Панели управления администратора

### 🌟 Что нового?

Мы продолжаем расширять возможности кастомизации HubEx! Теперь в Панели управления администратора можно добавлять собственные виджеты для отображения ключевой информации и быстрого доступа к важным функциям.

### 📊 Ключевые возможности

#### 🎯 Встраивание виджетов

* Добавление пользовательских виджетов в левую колонку Панели управления
* Поддержка **нескольких виджетов** одновременно
* Автоматическое расположение виджетов друг под другом

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges09092025/j9YcWc6h_image.png)

### 🛠️ Техническая реализация

* **Стандартный размер**: высота 294px (как блок "Информация о тенанте")
* **Адаптивная ширина**: соответствует ширине левой колонки
* **Безопасное подключение**: автоматическая передача access\_token

#### 🔄 Простой процесс подключения

1. Система проверяет доступные виджеты через API
2. Для виджета "Подпункт в Панель управления" (ID: 29) создается iframe
3. Автоматический запрос ссылки для виджета
4. Отображение контента с передачей токена доступа

### 🎨 Пример использования

#### Возможные сценарии:

* 🔔 **Уведомления** - важные оповещения для администратора
* ⚡ **Быстрый доступ** - ссылки на часто используемые функции
* 📋 **Сводки** - краткая информация о состоянии системы

### ⚙️ Как начать использовать?

1. Для добавления плагина в систему необходимо обратиться в поддержку Telegram [@hubex\_bot](https://t.me/hubex_bot) или на почту [help@hubex.ru](mailto:help@hubex.ru)
2. Далее, добавляете в тенант созданный плагин
3. Система автоматически добавляет новый пункт меню в административной панели
4. При клике открывается iframe со ссылкой плагина

### 🔒 Безопасность

* Все виджеты работают через защищенное соединение
* Автоматическая передача токена доступа
* Изоляция контента через iframe
