#### Массовые действия для списка Заявок: обновление от 31.05.2023

Мы выпустили новые функции для оптимизации рабочих процессов и улучшения взаимодействия пользователей с HubEx!

##### Массовое назначение исполнителей на Заявки

Возможность массово назначать исполнителей для **Заявок** — полезная функция, которая поможет эффективно управлять организацией работы ваших сотрудников. Теперь можно сразу для нескольких **Заявок** назначать конкретных технических специалистов.

Как это работает?

Массовое назначение исполнителей доступно для карточного и табличного отображения списка заявок в меню **Заявки - Все заявки**.

С помощью фильтров или (и) ручного выбора сформируйте список заявок для назначения сотрудников. Выбрать **Заявки** вручную можно с помощью установки флажков слева.

Чтобы выбрать все **Заявки** на странице, можно установить один флажок в панели редактирования (над списком).

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/1.png)

Чтобы назначить исполнителей для выбранных **Заявок**, нажмите кнопку **Редактировать**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/2.png)

В форме редактирования для параметра **Исполнитель** выберите подходящих сотрудников и примените изменения.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/3.png)

Назначенные исполнители сразу же отобразятся для каждой из выбранных **Заявок**.

Таким образом можно не только назначать исполнителей, но и заменять исполнителей, ранее назначенных на **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/4.png)

Подробнее читайте в статье нашей базы знаний: [Массовые действия для списка заявок](https://wiki.hubex.ru/docs/FAQ/RU/user/GroupActionsForTickets.html.md).

##### Массовое удаление Заявок

Благодаря новому функционалу пользователи могут легко удалять ненужные или неактуальные **Заявки** в web-приложении HubEx.

Как это работает?

Массовое удаление доступно для карточного и табличного отображения списка заявок в меню **Заявки - Все заявки**.

С помощью фильтров или (и) ручного выбора сформируйте список заявок, которые необходимо удалить. Выбрать **Заявки** вручную можно с помощью установки флажков слева.

Чтобы выбрать все **Заявки** на странице, можно установить один флажок в панели редактирования (над списком).

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/5.png)

Для удаления выбранного списка нажмите кнопку **Удалить** на панели редактирования.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/GroupActions31052023/6.png)

Предложения по доработкам в системе и сообщения об ошибках просим отправлять к нам в "чат с поддержкой" в веб-приложении или на почту  [help@hubex.ru](mailto:help@hubex.ru) .
