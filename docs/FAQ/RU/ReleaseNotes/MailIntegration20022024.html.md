#### Интеграция по электронной почте через пересылку: обновление от 20.02.2024

Новый и более легкий способ настройки интеграции через электронную почту уже доступен в HubEx. Встречайте: интеграция через пересылку!

#### Прием заявок через пересылку

Как вы уже знаете, в HubEx уже существует тонкая настройка интеграции с электронной почтой через настройки imap, создание регулярных выражений и различных шаблонов.

Наша команда предлагает облегченный вариант настройки интеграции, который не займет у пользователей много времени.

Интеграция через пересылку предполагает настройку переадресации писем для вашего почтового ящика на специальный ящик, созданный индивидуально для бизнес-аккаунта HubEx. **Заявки** будут создаваться по единому шаблону для тех писем, для которых будет настроена пересылка.

Как это работает?

В меню консоли администрирования в разделе **Интеграция - Прием заявок по эл.почте** вы найдете адрес специального ящика, созданного для вашего бизнес-аккаунта. Название ящика необходимо скопировать и вставить его в специальном разделе вашего почтового сервиса.

По щелчку на подсказку **Как настроить пересылку?** можно открыть краткие подсказки по настройке интеграции. Для разных почтовых сервисов настройки будут немного отличаться.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketMail/Integration2.jpg)

В статье нашей базы знаний [Интеграция по электронной почте, подача заявок по электронной почте](https://wiki.hubex.ru/docs/FAQ/RU/admin/TicketMail.html.md) мы подробно рассказываем, как настроить пересылку для ящиков Gmail и Yandex. После настройки интеграции по пересылке письма будут трансформироваться в **Заявки** системы HubEx.

После настройки пересылки заказчики смогут отправлять письма на ваш электронный адрес сервиса Gmail или Yandex.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketMail/Mail4.jpg)

А в HubEx такие письма будут приходить, как **Заявки**, которые при необходимости можно редактировать.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/TicketMail/Integration21.jpg)
