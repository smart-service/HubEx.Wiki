## 🔗 Гиперссылки в календаре для быстрого перехода к заявкам

Больше не нужно искать заявки вручную — всё открывается одним нажатием!

## ✅ Новый раздел «Чек-листы» в истории изменений заявок

История изменений теперь фиксирует и работу с чек-листами!

В систему добавлено логирование:

* добавления чек-листа к заявке;
* отметок пунктов (чек-боксов);
* заполнения дополнительных атрибутов пунктов (текст, число, файл и др.).

Теперь вы всегда сможете увидеть, **когда и кем были заполнены пункты чек-листа**, что делает контроль качества и прозрачность выполнения заявок ещё выше.

## 🔍 Фильтрация заявок по ссылкам (query-параметры)

Теперь вы можете **переходить по ссылкам сразу в нужный список заявок с готовыми фильтрами**!

Поддерживается:

* переход из аналитических отчётов в список заявок по выбранной стадии или параметру;
* переход из дочерних заявок к связанным;
* возможность **делиться ссылками** на уже отфильтрованные списки с коллегами.

Это ускоряет навигацию и облегчает совместную работу с заявками.

## 📱 Офлайн-режим в мобильном приложении: теперь можно редактировать заявки без интернета

Мы рады сообщить о значительном расширении функционала офлайн-режима. Теперь вы можете редактировать заявки даже без подключения к интернету, а все изменения автоматически синхронизируются при появлении сети.

### 🚀 Что нового?

**Расширенный офлайн-режим теперь поддерживает:**

**Основные поля заявки:**

* Поле описание

**Дополнительные поля:**

* Числовые поля (целые и дробные числа)
* Поля с датой и временем
* Текстовые поля (однострочные и многострочные)
* Переключатели (да/нет)
* Поле с типом Рисунок

### 💡 Как это работает?

1. **Редактируйте в офлайне** — вносите изменения в заявки даже без интернета
2. **Автоматическая очередь** — система сохраняет все изменения в очередь отправки
3. **Синхронизация при подключении** — как только появляется сеть, данные автоматически отправляются на сервер
4. **Без потери данных** — все изменения гарантированно сохраняются

### 🛠️ Особенности реализации:

**Права доступа:**

* Редактирование доступно согласно вашим правам на текущей стадии заявки
* При переводе заявки на следующую стадию в офлайне сохраняются права редактирования

**Специальные функции:**

* Полная поддержка сканирования NFC-карт (дополнительные поля с названием `_NFC`)
* Работа со всеми типами полей даты и времени
* Загрузка рисунков (подпись)

### 🎯 Преимущества:

✅ **Непрерывность работы** — работайте в любых условиях, даже без интернета
✅ **Автоматическая синхронизация** — не нужно вручную отправлять изменения
✅ **Широкий охват полей** — редактируйте практически любые данные заявки
✅ **Надежность** — гарантированное сохранение всех внесенных изменений

### 📌 Как использовать?

1. Откройте заявку в мобильном приложении
2. Внесите необходимые изменения (даже без интернета)
3. Закройте заявку — система сохранит изменения в очередь
4. При появлении сети данные автоматически отправятся на сервер
