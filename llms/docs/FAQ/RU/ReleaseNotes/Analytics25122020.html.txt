#### Отчет Общая аналитика: обновление от 25.12.2020

Информация о показателях вашего бизнеса теперь представлена в удобном формате в разделе **Аналитика - Общая аналитика**. Анализируйте готовые данные и принимайте верные решения вместе с HubEx.

##### Показатели отчета Общая аналитика

Отчет **Общая аналитика** состоит из нескольких разделов:

* ключевые показатели представлены наверху панели: они помогут оценить среднее время выполнения и закрытия заявок, коэффициент загрузки исполнителей, количество заявок, выполняемых одним исполнителем и средний рейтинг;
* графики и диаграммы помогут установить, успевает ли сервисная служба выполнять заявки, как меняется динамика поступающих заявок по сравнению с предыдущим периодом в разрезе типов заявок, видов работ, критичности.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/1.png)

Справа находится вертикальная кнопка **Фильтры**. С помощью фильтров можно задать любой период для анализа данных и сделать отбор по различным разрезам: типам заявки, видам работ, объектам, компаниям и т.д.

Верхний левый блок представляет выбор одного из трех вариантов аналитики: **Поступивших заявок**, **Закрытых заявок**, **Незакрытых заявок** (количество указывается на момент окончания указанного в фильтрах периода). В зависимости от выбранного варианта будут изменяться диаграммы и графики аналитики.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/2.png)

У каждого графика есть информационная подсказка, в которой описан принцип расчета, работы с графиками и даны рекомендации по оценке показателей.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/3.png)

##### Аналитические разрезы по заявкам

Инструмент **Аналитические разрезы по заявкам** представляет собой дерево, которое детализируется в различных разрезах.

Для удобной работы с отчетами пользуйтесь функцией **Режим фокусировки**. Она доступна для каждого графика и диаграммы.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/4.png)

Добавляйте новые разрезы по кнопке +.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/5.png)

Например, вы хотите увидеть закрытые заявки по конкретным компаниям, объектам, критичности, видам работ, исполнителям, рейтингам. При отображении информации по выбранным разделам внизу отражается количество заявок.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/6.png)

Для каждого разреза доступна функция детализации до списка заявок. Нажмите на любой раздел правой кнопкой мыши и выберите пункт меню Детализация - Список заявок. Из списка можно будет открыть любую заявку.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/7.png)

##### Диаграммы соотношений

Диаграммы соотношений отражают аналитику в разрезе типов заявок, критичности и видов работ. При наведении мыши на любой сектор диаграммы, открываются детальные графики динамики и список заказчиков, по которым показатели лидируют.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/Analytics25122020/8.png)

Подробнее об отчете читайте в статье базы знаний: [Общая аналитика](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/GeneralAnalytics.html.txt).
