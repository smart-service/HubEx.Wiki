#### Форматирование поля Описание заявки: обновление от 23.01.2023

Улучшения для поля **Описание заявки**уже в системе! При описании заявки теперь можно выделять главное, оформлять списки, прикреплять гиперссылки.

Обновление доступно для web версии и мобильных приложений **HubEx исполнитель** и **HubEx заказчик**.

##### Новый формат описания заявки для web портала HubEx

В меню **Заявки - Все заявки** при создании новой **Заявки** или редактировании уже созданных обратите внимание на поле **Описание заявки**. Панель для работы с текстом доступна сверху поля.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DescriptionField23012023/7WFj4V7_Screenshot-9.png)

Теперь в описание заявки можно вставлять текст, скопированный из сторонних систем и документов. Для текста доступно редактирование: выделение жирным шрифтом, курсивом, подчеркнутым шрифтом, изменение размера шрифта, оформление нумерованных и ненумерованных списков, прикрепление гиперссылок и изменение цвета шрифта.

##### Новый формат описания заявки для мобильных приложений HubEx

В мобильных приложениях **HubEx исполнитель** и **HubEx заказчик** также доступна форма редактирования текста.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DescriptionField23012023/Ks8LrVx_Microsoft-Teams-image-2.jpg)

Текст в описание заявки можно скопировать из других приложений, сообщений и отформатировать по кнопке **Редактировать**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DescriptionField23012023/bvs7X9K_Microsoft-Teams-image-1.jpg)

Для скачивания приложений отсканируйте подходящий QR-код ниже.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DescriptionField23012023/4fN1c7r_QR.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/DescriptionField23012023/Db13PBn_QR.jpg)
