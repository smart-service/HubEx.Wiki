### **🔐 Новая система управления доступом к плагинам в ролях пользователей**

Уважаемые администраторы HubEx!

Мы представляем новую гибкую систему управления доступом к плагинам, которая позволит вам точно настраивать права доступа для разных ролей пользователей.

### **🔌 Что такое плагины в HubEx?**

**Плагин** — это дополнительный функционал, который встраивается в систему через **iframe** и расширяет возможности HubEx под ваши уникальные бизнес-задачи.

**Примеры плагинов:**

* Кастомные отчеты и дашборды
* Дополнительные печатные формы (акты, счета)
* Специализированные разделы в заявках
* Дополнительные вкладки в мобильном приложении

**Как работают плагины:**

* Отображаются в интерфейсе как обычные элементы системы
* Работают в защищенном режиме через iframe
* Могут быть подключены только к определенным разделам (заявки, объекты, отчеты и т.д.)

### 🚀 Что нового?

**Автоматическая группа полномочий для плагинов:**

* В настройках ролей появилась новая группа **"Плагины"**
* Каждый плагин теперь отображается как отдельное полномочие
* Автоматическое обновление списка при добавлении/удалении плагинов в тенанте

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/PluginManagement.png)

### 💡 Как это работает?

**Динамическое управление плагинами:**

* **Новый плагин добавлен** → автоматически появляется в настройках всех ролей (выключен по умолчанию)
* **Плагин удален** → автоматически убирается из настроек ролей
* **Гибкая настройка** → для каждой роли можно включать/выключать отдельные плагины

**Формат отображения:**

* Название формируется как: **"[Название плагина] - [Место размещения]"**
* Пример: "Акт выполненных работ - Подпункт в форме заявки"

### 🎯 Преимущества для администраторов:

**Точный контроль** — включайте только нужные плагины для каждой роли
 **Автоматизация** — система сама обновляет список при изменениях
 **Понятный интерфейс** — четкие названия и группировка
 **Гибкость** — разные наборы плагинов для разных ролей

### 📌 Как настроить?

1. Перейдите в настройки роли (**Администрирование → Роли**)
2. Откройте вкладку **"Плагины"**
3. Включите/выключите нужные плагины для этой роли
4. Сохраните изменения
