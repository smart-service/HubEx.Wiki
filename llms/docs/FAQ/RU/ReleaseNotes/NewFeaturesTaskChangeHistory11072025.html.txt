## 🚀 Улучшенная история изменений с геоданными и новыми возможностями от 11.07.2025

---

## 🌟 Что нового?

Мы доработали функционал истории изменений заявок! Теперь вы можете:

### 📍 Отслеживание местоположения

* Видеть где находился исполнитель при внесении изменений (в мобильном приложении)
* Три статуса местоположения:
  + На объекте
  + Вне объекта
  + Не определено
* Кликабельные маркеры для просмотра на карте

### 🗂️ Удобное представление данных

* Новое боковое окно с детализированной информацией
* Предпросмотр вложенных файлов
* Отображение исполнителей с фотографиями
* Кликабельные ссылки на связанные заявки (Родительская заявка)

## 📌 Основные изменения

### 1. Новый столбец "Местоположение"

| **Система** | **Статус** | **Отображение** |
| --- | --- | --- |
| МП Исполнителя | На объекте | (кликабельно) |
| МП Исполнителя | Вне объекта | (кликабельно) |
| МП Исполнителя | Не определено |  |

### 2. Интерактивная карта

* При клике на маркер открывается карта с:
  + Местоположением объекта
  + Точкой, где находился исполнитель
  + Маршрутом между ними

### 3. Боковое окно изменений

* Полная информация о изменении
* Предпросмотр вложений (документы, изображения)
* Фотографии и данные исполнителей
* Ссылки на связанные заявки

## ⚠️ Важные нюансы

* Данные о местоположении доступны только для действий из МП Исполнителя
* Для работы карты требуются корректные координаты объекта
