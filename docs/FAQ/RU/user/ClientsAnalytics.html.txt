#### Отчет для клиента

В этом разделе вы научитесь работать с **Отчетом для клиента**, который содержит в себе:

* [Ключевые показатели по заявкам;](https://wiki.hubex.ru/docs/FAQ/RU/user/ClientsAnalytics.html.txt#kpi)
* [Визуализации по выполненным заявкам и SLA;](https://wiki.hubex.ru/docs/FAQ/RU/user/ClientsAnalytics.html.txt#diagramms)
* [Таблицу со списком активных заявок.](https://wiki.hubex.ru/docs/FAQ/RU/user/ClientsAnalytics.html.txt#table)

В HubEx представлены аналитические отчеты реализованные на платформе Power BI. С помощью бизнес-аналитики вы
можете анализировать данные, получать ценные сведения для принятия бизнес-решений.

Каждый аналитический отчет реализован в виде панели с различными графиками, диаграммами, сводными
таблицами и показателями.

**Отчет для клиента** в HubEx дает возможность получить сводные данные и основные показатели по **Заявкам**
в разрезе ваших клиентов (заказчиков). Данный отчет вы сможете предоставлять своим клиентам в качестве оперативной сводки по
состоянию выполнения **Заявок**. Также данный отчет поможет клиентам оценить частоту создания **Заявок** в разрезе
определенных **Объектов**, что позволит принять решение об изменении обслуживания, замене часто
выходящего из строя оборудования и т.д.

Представление панели отчетов основывается на данных в **Заявках**. С помощью параметров фильтрации (панель справа) вы можете выбрать организацию
(компанию-заказчика), сделать отбор по одному или нескольким **Объектам** заказчика, а также по
компаниям-подрядчикам, которые выполняют с **Заявки** клиентов. Подрядчиком может являться в том числе и ваша организация.

Данные в отчете выводятся за период, выбранный на панели фильтрации.

Список **Заявок** (таблица внизу) и визуализации изменяются в зависимости от выбора рассматриваемых заявок на панели фильтрации:

* **Активные заявки**: в отчете будет отображаться таблица со списком активных **Заявок** за выбранный период, и две первых визуализации будут отражать данные по активным **Заявкам** (**Активные заявки по типу заявки/критичности** и **Активные заявки по типу оборудования**);
* **Выполненные заявки**: в отчете будет отображаться таблица со списком выполненных **Заявок** за выбранный период, и две первых визуализации будут отражать данные по выполненным **Заявкам** (**Выполненные заявки по типу заявки/критичности** и **Выполненные заявки по типу оборудования**);
* **Созданные заявки**: в отчете будет отображаться таблица со списком всех созданных **Заявок** за выбранный период, и две первых визуализации будут отражать данные по созданным **Заявкам** (**Созданные заявки по типу заявки/критичности** и **Созданные заявки по типу оборудования**).

Далее в статье визуализации будут рассмотрены на примере **Выполненных заявок**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/ClientsAnalitics2.jpg)

##### Ключевые показатели по заявкам

В верхней части отчета находятся основные показатели (показатели отображаются за выбранный период):

* **Выполнено заявок** - это количество выполненных **Заявок**;
* **Просрочено** - это количество активных (открытых) и выполненных **Заявок** (которые были выполнены после крайнего срока закрытия), у которых истек крайний срок закрытия;
* **Непринятые заявки** - количество **Заявок**, на которые не был назначен исполнитель;
* **Заявок на согласовании** - количество **Заявок** на согласовании у заказчика (клиента);
* **Без срока закрытия** - количество **Заявок**, в которых не был установлен параметр **Крайний срок закрытия**. Подробнее читайте в статье: [Создание заявки](https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingTicket.html.txt);
* **Выполнено работ, в часах** - это общее количество времени выполнения **Заявок** (параметр считается часам, указанным в выполненных работах на вкладке **Выполнение** в **Заявке**).

##### Визуализации по выполненным заявкам и SLA

Первые две визуализации изменяются в зависимости от выбора рассматриваемых заявок: **Активные заявки**, **Выполненные заявки**, **Созданные заявки**. В данном пункте визуализации будут рассмотрены на примере **Выполненных заявок**.

Визуализация **Выполненные заявки по типу заявки/критичности** показывает соотношение выполненных **Заявок** по **Типам заявок**. При
нажатии на любой из секторов вы перейдете к диаграмме соотношений выполненных **Заявок** выбранного типа по
критичности. Вернуться к исходной визуализации можно по щелчку в центр диаграммы.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/TypeCritical.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/TypeCritical2.jpg)

Визуализация **Выполненные заявки по типу оборудования** представляет собой количественные показатели выполненных **Заявок** в разрезе **Типов
оборудования**. **Типы оборудования** создаются в [консоли
администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.txt) и устанавливаются в карточках **Объектов**. Подробнее
читайте в статьях: [Создание обслуживаемого
оборудования](https://wiki.hubex.ru/docs/FAQ/RU/user/CreatingObjects.html.txt), [Типы оборудования](https://wiki.hubex.ru/docs/FAQ/RU/admin/ObjectsType.html.txt).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/Diagram.jpg)

Данные для визуализации **SLA подрядных организаций** рассчитываются как отношение количества закрытых с
просрочкой **Заявок** к общему количеству закрытых **Заявок** за предыдущий месяц.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/Diagram2.jpg)

##### Таблица со списком выполненных заявок

Таблица изменяется в зависимости от выбора рассматриваемых заявок: **Активные заявки**, **Выполненные заявки**, **Созданные заявки**. В данном пункте таблица будет рассмотрена на примере **Выполненных заявок**.

В таблице **Список заявок** отражаются все выполненные **Заявки**. Вы можете открыть любую **Заявку** по ссылке из крайней колонки слева.

Также **Список заявок** можно отобразить в виде карты: выполненные **Заявки** будут располагаться на карте.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ClientsAnalitics/list.jpg)
