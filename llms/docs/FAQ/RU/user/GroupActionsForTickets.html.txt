#### Массовые действия для списка заявок

В этом разделе вы узнаете:

* [Как назначить исполнителей на несколько Заявок](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/GroupActionsForTickets.html.txt#performer);
* [Как удалить несколько Заявок](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/GroupActionsForTickets.html.txt#deleting).

С помощью массовых действий над списком заявок можно быстро добавить/изменить определенные параметры сразу в
нескольких **Заявках**:

* заполнить/заменить **Исполнителя**;
* удалить **Заявки**.

##### Как назначить исполнителей на несколько Заявок

Откройте список всех **Заявок** в меню **Заявки - Все заявки**.

Обратите внимание! Массовые действия доступны для карточного и табличного представления списка. Подробно о форматах представления списка читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/Filters.html.txt).

Чтобы назначить исполнителей на несколько **Заявок**, необходимо:

1. С помощью фильтров сделайте выбор группы **Заявок**, для которых необходимо назначить исполнителя. Например, необходимо
   назначить исполнителя для **Заявок** от компании-заказчика **Компания Cook Kitchen**. Поэтому с помощью фильтров отбираем
   все **Заявки** с **Заказчиком = Компания Cook Kitchen**. Таким образом получим отфильтрованный список.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Tickets.jpg)

   Использование фильтров целесообразно при большом количестве **Заявок** в списке для того, чтобы минимизировать
   ошибку выбора их вручную. Данный шаг не является обязательным. Подробнее о работе с фильтрами читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/Filters.html.txt).
2. Теперь нужно включить режим массовых действий. Наведите курсором мышки слева от
   любой **Заявки** из отфильтрованного списка. Вы увидите, как появится окошко для установки флажка выбора.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectEmpty.jpg)

   Установите флажок. Сверху появится панель редактирования. Выбрать все **Заявки** на странице можно с помощью
   установки флажка в панели редактирования. Либо продолжить выбор **Заявок** с помощью установки флажков вручную

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectedTick.jpg)
3. Нажмите кнопку **Редактировать**. Откроется форма для выбора параметров. Чтобы назначить исполнителей, нажмите кнопку
   **+ Изменить**.

   Обратите внимание! С помощью данного функционала можно назначать исполнителей как на **Заявки**, где исполнитель еще не был указан, так и на **Заявки**, где уже был выбран исполнитель(ли).

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Edit.jpg)
4. Выберите специалистов, которых нужно назначить на **Заявки** и нажмите кнопку **Выбрать**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/ParamsSelect.jpg)
5. Нажмите кнопку **Применить**.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/ParamsSelect2.jpg)

Назначенный исполнитель(ли) сразу отобразятся для выбранных **Заявок**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Performer.jpg)

##### Как удалить несколько Заявок

Откройте список всех **Заявок** в меню **Заявки - Все заявки**.

Обратите внимание! Массовые действия доступны для карточного и табличного представления списка. Подробно о форматах представления списка читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/Filters.html.txt).

Чтобы удалить несколько **Заявок**, необходимо:

1. С помощью фильтров сделать выбор группы **Заявок**, которые необходимо удалить. Например, необходимо
   удалить **Заявки** с видом работ **Гарантийный ремонт**. Поэтому с помощью фильтров отбираем
   все **Заявки** с **Видом работ = Гарантийный ремонт**. Таким образом получим отфильтрованный список.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Tickets2.jpg)

   Использование фильтров целесообразно при большом количестве **Заявок** в списке для того, чтобы минимизировать
   ошибку выбора их вручную. Данный шаг не является обязательным. Подробнее о работе с фильтрами читайте в статье: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/Filters.html.txt).
2. Теперь нужно включить режим массовых действий. Наведите курсором мышки слева от
   любой **Заявки** из отфильтрованного списка. Вы увидите, как появится окошко для установки флажка выбора.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectEmpty2.jpg)

   Установите флажок. Сверху появится панель редактирования. Выбрать все **Заявки** на странице можно с помощью
   установки флажка в панели редактирования. Либо продолжить выбор **Заявок** с помощью установки флажков вручную

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/SelectedTick2.jpg)
3. Нажмите кнопку **Удалить**. Откроется диалоговое окно для подтверждения удаления.

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Deleting.jpg)

   ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/Deleting2.jpg)

Удаленные **Заявки** сразу исчезнут из списка.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/TicketList.jpg)

Найти и просмотреть удаленные **Заявки** можно с помощью фильтров: выбрать в разделе **Видимость заявок** вариант **Только удаленные**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/TicketList2.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/GroupActionsForTickets/TicketList3.jpg)
