#### Фильтрация заявок по дополнительным полям с числовым форматом: обновление от 01.12.2023

Новые возможности для работы с дополнительными полями уже доступны в HubEx! Теперь дополнительные поля числового формата доступны для фильтрации **Заявок**.

Фильтровать и сортировать данные можно на основе определенных числовых критериев, а также числовых диапазонов.

Это позволит вам еще более эффективно использовать дополнительные поля в системе и находить необходимую информацию.

##### Фильтрация заявок по дополнительным полям

Вы уже знаете, что в HubEx список **Заявок** можно фильтровать, используя текстовые дополнительные поля. Теперь в рамках этого функционала для поиска доступны дополнительные поля с форматом **Целое число** и **Дробное число**.

Дополнительные поля создаются и настраиваются в консоли администратора. Подробнее об этом читайте в статье: [Создание дополнительных полей](https://wiki.hubex.ru/docs/FAQ/RU/user/AdditionalFields.html.md)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterAddFields/Format.jpg)

В меню **Заявки - Все заявки** на панели фильтров внизу будут доступны дополнительные поля, созданные для **Заявок**.

Для числовых форматов дополнительных полей доступен выбор настроек (**Равно**, **Диапазон**, **Больше чем** и **Меньше чем**). Таким образом вы сможете быстро и эффективно находить **Заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterAddFields/Filter.jpg)

Задайте необходимы параметры и значения поиска для числовых полей и нажмите кнопку **Применить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterAddFields/Filter2.jpg)

Список **Заявок** будет отфильтрован согласно установленным параметрам поиска по дополнительным полям.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/FilterAddFields/Result.jpg)

Дополнительно о работе с фильтрами для **Заявок** читайте в статье нашей базы знаний: [Список заявок, поиск заявок с помощью фильтров](https://wiki.hubex.ru/docs/FAQ/RU/user/Filters.html.md).

Предложения по доработкам в системе и сообщения об ошибках просим отправлять к нам в "чат с поддержкой" в веб-приложении или на почту  [help@hubex.ru](mailto:help@hubex.ru) .
