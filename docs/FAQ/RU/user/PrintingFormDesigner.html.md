## Что такое Конструктор печатных форм

**Конструктор печатных форм** — это отдельный раздел в системе HubEx, позволяющий создавать шаблоны документов (печатных форм) для заявок.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/боковое меню - пункт конструктор печатных форм.jpg)

Шаблоны состоят из текста, таблиц и специальных переменных, которые автоматически подставляют данные из конкретной заявки, объекта, компании и других связанных сущностей.

## **Ограничения бесплатного конструктора печатных форм**

Наш конструктор печатных форм позволяет быстро и удобно создавать базовые формы для печати без привлечения разработчиков. Однако важно понимать его текущие ограничения, чтобы оценить, подходит ли он для ваших задач.

**Что доступно в текущей версии конструктора:**

* **Использование переменных:** Вы можете добавлять в макет любые данные из системы, используя доступный набор переменных (например,{{Тип заявки}}, {{Компания-заказчик | Название компании}}).
* **Стандартные таблицы:** Вставка предопределенных таблиц для отображения таких данных, как:

  + Чек-листы

    ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/checklist.png)
  + Список выполненных работ

    ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/ComplitedWorks.png)

**Какие сложные задачи пока не поддерживаются:**
Конструктор не предназначен для создания форм со сложной бизнес-логикой. Если ваша печатная форма требует следующего, она не может быть реализована через интерфейс конструктора:

* **Кастомизируемые таблицы:** Создание таблиц с произвольной структурой столбцов, данными не из стандартного набора или особыми правилами форматирования.
* **Логика вычислений:** Реализация расчетов на основе данных из формы (например, вычисление итоговой суммы, скидок, налогов на лету прямо в печатной форме).
* **Условное отображение блоков:** Динамическое скрытие или показ частей документа в зависимости от значений определенных полей (например, "показывать этот блок только если тип заказа - 'Ремонт'").

**Что делать, если возможностей конструктора недостаточно?**
Если вам необходима печатная форма с перечисленной выше сложной логикой, пожалуйста, обратитесь к вашему менеджеру в HubEx или через сайт [HubEx](https://hubex.ru/services/automate). Наша команда разработки сможет реализовать для вас кастомную печатную форму в рамках индивидуальных доработок.

## Как создать форму

1. Перейдите в раздел **Конструктор отчетов** (отдельный пункт меню в системе).
![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/Конструктор печатных форм.jpg)2. Нажмите кнопку **«+»** для создания новой печатной формы.
3. Заполните поля:
   * **Название** — отображается как название вкладки в заявке;
   * **Описание** — краткое пояснение к форме (необязательно);
4. В текстовом редакторе введите содержимое шаблона:
   * Статический текст и форматирование (жирный, списки, таблицы и пр.);
   * Переменные, которые подставят данные из заявки. Переменные обязательно указывайте в двойных фигурных скобках, например: {{Номер}}.
5. Для удобства добавления переменных используйте выпадающие списки переменных вверху конструктора.
6. Обратите внимание:
   * Переменная при добавлении вставляется в конец формы — её нужно вручную переместить в нужное место внутри шаблона.
   * Время, отображаемое в переменных с датой и временем, подставляется с учетом часового пояса тенанта (по умолчанию).
7. Нажмите **«Сохранить»** для создания шаблона.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/Форма создания печатной формы.jpg)

## Как удалить форму

1. В списке печатных форм найдите нужную.
2. Нажмите на кнопку с тремя точками рядом с формой.
3. Выберите **«Удалить»**.
4. Подтвердите удаление.

После удаления форма полностью удаляется из системы и перестает отображаться в заявках.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/меню формы.jpg)

## Как деактивировать форму

1. В списке печатных форм нажмите кнопку с тремя точками у нужной формы.
2. Выберите **«Деактивировать»**.
3. Форма становится неактивной, то есть скрывается из списка форм в заявках, но сохраняется в системе.
4. Для повторной активации выберите **«Активировать»** в том же меню.

## Как форма добавляется в заявку

* После создания шаблона в Конструкторе отчетов у заявки появляется новая вкладка с названием, которое вы указали при создании шаблона.
* При открытии вкладки происходит динамическая подстановка значений переменных из заявки.
* Если в Конструкторе создано несколько шаблонов — для каждого создается своя вкладка.
![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/список актов в заявке.jpg)* В форме вместо переменных, значение которых отсутствует, будет пустая строка.
* Вкладка печатной формы содержит кнопку **«Распечатать»**, которая вызывает стандартное окно печати браузера.
![](https://wiki.hubex.ru/attachments/images/FAQ/USER/PrintingFormDesigner/кнопка печати.jpg)

## Как сохранить/распечатать форму

1. Откройте заявку и перейдите на вкладку печатной формы с нужным шаблоном.
2. Убедитесь, что все данные корректно подставлены.
3. Нажмите кнопку **«Распечатать»** вверху вкладки с формой.
4. Откроется стандартное окно печати вашего браузера.
5. Выберите нужный принтер и настройки печати.
6. Нажмите **«Печать»** для отправки документа на печать.
7. Если нужно, сохраните форму в PDF с помощью функций браузера.
8. Если в заявке несколько шаблонов, повторите процесс для каждого из них по отдельности.
