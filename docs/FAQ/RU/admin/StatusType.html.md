#### Статусы заявок

**Статус заявки** – это визуальное отображение **Стадии заявки** для мобильного приложения заказчика.
Для каждого статуса задается свой цвет.
**Статус заявки** привязывается к **Стадии заявки** при создании **Жизненного цикла заявки**. Подробнее читайте в статье: [Жизненный цикл заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/TicketLifeCycle.html.md).

Создать новый **Статус заявки** можно в [консоли
администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.md) в меню **Настройки заявки - Статусы заявки** по кнопке **Создать
статус заявки**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/StatusType/NewStatus.jpg)

Введите наименование статуса и выберите цвет, которым статус будет отображаться в мобильном приложении
заказчика. Нажмите кнопку **Сохранить**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/StatusType/MobCustomer.jpg)
