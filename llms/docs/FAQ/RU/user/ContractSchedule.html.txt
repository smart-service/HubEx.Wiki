#### График действия договоров

В HubEx представлены аналитические отчеты реализованные на платформе Power BI. С помощью бизнес-аналитики вы
можете анализировать данные, получать ценные сведения для принятия бизнес-решений.

Каждый аналитический отчет реализован в виде панели с различными графиками, диаграммами, сводными
таблицами и показателями.

**График действия договоров** в HubEx дает возможность отслеживать сроки окончания договоров и выявлять
**Заявки**, выполненные вне периода действия договоров.

Договоры создаются в карточке **Компании** в меню **Компании - Все компании**. Подробнее читайте в статье: [Создание обслуживаемых компаний: договор с клиентом](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/CreatingCompany.html.txt#company6).

**График действия договоров** находится в меню **Аналитика - График действия договоров**.
Данные
в отчете обновляются один раз в сутки. Дата обновления отображается вверху слева.

Для формирования графика необходимо выбрать период и при необходимости задать разбивку периода: квартал, месяц,
неделя, день.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ContractSchedule/Schedule.jpg)

С помощью панели фильтров справа можно выбрать конкретного **Заказчика**, уточнить **Тип
договора** и так далее.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ContractSchedule/Filter.jpg)

**Заявки**, выполненные вне срока действия договора отображаются в таблице значком флажка. Чтобы открыть
список таких
**Заявок**, необходимо щелкнуть на ячейку с флажком и нажать кнопку **Открыть список
заявок** на панели справа (под кнопкой
**Фильтров**).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ContractSchedule/Ticket.jpg)

В открывшейся таблице будет находиться список **Заявок**. **Заявку** можно открыть в соседнем окне по нажатию
на соответствующий значок ссылки в колонке URL слева.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ContractSchedule/List.jpg)
