#### Экспорт

В этом разделе вы научитесь:

* [Экспортировать заявки;](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md#tickets)
* [Экспортировать медиафайлы из заявок;](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md#media)
* [Экспортировать объекты;](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md#objects)
* [Экспортировать пользователей;](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md#workers)
* [Экспортировать компании.](https://wiki.hubex.ru/docs/FAQ/RU/user/Export.html.md#companies)

С помощью функций экспорта вы можете выгрузить **Заявки**, **Объекты**, **Пользователей**
(**Сотрудников**, **Заказчиков**), **Компании** в формате таблиц Excel.

Функции экспорта и импорта в системе HubEx полностью совместимы с Excel, ExcelOnline, частично с Google Spreadsheet,
но
не
совместимы с Mac Spreadsheet. Также не рекомендуем использовать сторонние сервисы для работы с шаблонами Excel из-за
возможных ограничений
функционала.

Обратите внимание! Для **Заявок**, **Объектов**, **Пользователей** (**Сотрудников**
и **Заказчиков**) и **Компаний** вы можете настроить
фильтрацию и экспортировать только отобранные элементы. Фильтры доступны в правой части справочников. Подробнее
читайте в статье: [Список заявок, поиск заявок с
помощью фильтров](https://wiki.hubex.ru/docs/FAQ/RU/user/Filters.html.md).

##### Экспорт заявок

Экспорт заявок находится в меню **Заявки - Все заявки**. Чтобы начать экспорт, нажмите кнопку **Экспорт
заявок**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportTickets.jpg)

В появившемся окне вы можете выбрать один из вариантов экспорта:

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportVariant.jpg)

1. **Обычный экспорт** позволяет выгрузить минимальный набор основных данных по
   **Заявке**: **Номер**, **Тип заявки**, **Описание заявки**,
   **Оборудование**, **Вид работ**, **Крайний срок закрытия**, **Критичность**,
   **Исполнитель** и т.п. По нажатию на кнопку **Экспорт** начнется
   автоматическая выгрузка данных полей в таблицу
   Excel.
2. **Расширенный экспорт** позволяет пользователю самостоятельно выбрать набор выгружаемых полей.
   После выбора
   необходимых для выгрузки полей нажмите кнопку **Экспорт** для автоматической выгрузки в таблицу
   Excel. Выбранные в
   расширенном экспорте поля локально сохраняются на вашем компьютере. То есть при повторном экспорте
   у вас будут выбраны те поля, которые были выбраны при предыдущем экспорте **Заявок**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/Export.jpg)

Обратите внимание! Настройки экспорта заявок фиксируются за пользователем для текущего браузера и компьютера пользователя: запоминается вариант экспорта (Обычный экспорт, Расширенный экспорт), а также выбор полей для расширенного экспорта.

##### Экспорт медиафайлов из заявок

Экспорт медиафайлов предполагает выгрузку всех файлов (текстовых, фото и др.) из **Заявок**, а также чек-листов и выполненных работ (вкладки **Заявка**, **Чек-листы**, **Выполнение**). Нажмите кнопку экспорта в списке заявок и выберите вариант **Экспорт вложений**, затем нажмите **Экспорт**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/MediaExport.jpg)

Выгрузка осуществляется в отдельный архив, внутри которого находятся отдельные папки по каждой **Заявке** с номером и наименование оборудования.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/MediaExport2.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/MediaExport3.jpg)

Внутри каждой папки будут находиться папки с файлами из самой **Заявки** (вкладка **Заявка**), **Чек-листов** и (или) **Выполненных работ**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/MediaExport4.jpg)

Обратите внимание! Для экспорта медиафайлов также действуют настройки фильтров.
Вы можете отобрать определенные **Заявки** и выгрузить медиафайлы только по ним.

##### Экспорт объектов

Экспорт объектов находится в меню **Объекты|Оборудование - Объекты|Оборудование**. Чтобы начать экспорт, нажмите кнопку **Экспорт
объектов и оборудования**

В появившемся окне вы можете выбрать один из вариантов экспорта:

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportObject.jpg)

1. **Обычный экспорт** позволяет выгрузить минимальный набор основных данных по
   **Объектам**: **Название**, **Компания**, **Тип**,
   **Участок**, **Вид работ**, **Родительский элемент**, **Адрес** и т.п. По нажатию на кнопку **Экспорт** начнется
   автоматическая выгрузка данных полей в таблицу
   Excel.
2. **Расширенный экспорт** позволяет пользователю самостоятельно выбрать набор выгружаемых полей.
   После выбора
   необходимых для выгрузки полей нажмите кнопку **Экспорт** для автоматической выгрузки в таблицу
   Excel. Выбранные в
   расширенном экспорте поля локально сохраняются на вашем компьютере. То есть при повторном экспорте
   у вас будут выбраны те поля, которые были выбраны при предыдущем экспорте **Объектов**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportObject2.jpg)

По нажатию на кнопку экспорт автоматически начнется выгрузка списка объектов в формате Excel в папку загрузок на вашем локальном компьютере.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportObject3.jpg)

##### Экспорт пользователей

Экспорт пользователей предполагает раздельный импорт **Сотрудников** и **Заказчиков**.
Находится в меню **Пользователи -
Сотрудники** и в меню **Пользователи - Заказчики**. По нажатию на кнопку **Экспорт
сотрудников** или **Экспорт заказчиков** автоматически начнется выгрузка данных в таблицу
Excel.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportUser.jpg)

##### Экспорт компаний

Экспорт компаний находится в меню **Компании**. По нажатию на кнопку **Экспорт компаний**
автоматически начнется выгрузка данных в таблицу Excel.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/Export/ExportCompany.jpg)
