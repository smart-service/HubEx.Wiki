## Удаление компаний

**Способ 1.** Чтобы удалить существующую компанию:

1. Перейдите в раздел левого бокового меню **"Компании"** - подраздел **"Все компании"**;
2. Наведите курсор мыши на интересующую вас компанию. Справа, на карточке компании, отобразится "3 точки";
3. Нажмите на "3 точки";
4. Выберите "Удалить".

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew1.png)

**Способ 2.** Чтобы удалить существующую компанию:

1. Перейдите в раздел левого бокового меню **"Компании"** - подраздел **"Все компании"**;
2. Перейдите в карточку интересующей вас компании;
3. Справа наверху (возле кнопки **"Сохранить"**) нажмите на "3 точки";
4. Выберите "Удалить".

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew2.png)

**Обратите внимание!** В результате удаления компании, все привязанные к этой компании объекты будут сняты с публикации. На неопубликованные объекты нельзя подавать заявки.

**Обратите внимание!** Удаленные компании можно просмотреть в разделе левого бокового меню **"Компании"** - подраздел **"Все компании"** по параметру фильтра **"Видимость компаний"**.

**Обратите внимание!** Удаленную компанию нельзя восстановить. Вы можете создать только новую компанию по примеру старой компании.

## Удаление объектов/оборудования

**Способ 1.** Чтобы удалить существующий объект/оборудование:

1. Перейдите в раздел левого бокового меню **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"**;
2. Наведите курсор мыши на интересующий вас объект. Справа, на карточке объекта, отобразится "3 точки";
3. Нажмите на "3 точки";
4. Выберите "Удалить" или "Удалить с вложенными объектами".

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew3.png)

**Способ 2.** Чтобы удалить существующий объект/оборудование:

1. Перейдите в раздел левого бокового меню **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"**;
2. Перейдите в карточку интересующего вас объекта;
3. Справа наверху (возле кнопки **"Сохранить"**) нажмите на "3 точки";
4. Выберите "Удалить" или "Удалить с вложенными объектами".

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew4.png)

**Способ 3.** Чтобы удалить существующий объект/оборудование:

1. Перейдите в раздел левого бокового меню **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"**;
2. Наведите курсор мыши на интересующий вас объект. Слева выберите галочку на интересующих вас объектах;
3. Сверху нажмите на кнопку "Удалить".

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew5.png)

**Обратите внимание!** Нельзя удалять объекты, которые участвуют в активных заявах

**Обратите внимание!** Удаленные объекты можно просмотреть в разделе левого бокового меню HubEx **"Объекты|Оборудование"** - подраздел **"Объекты|Оборудование"** по параметру фильтра **"Видимость"**.

**Обратите внимание!** Удаленный объект нельзя восстановить. Вы можете создать только новый объект по примеру старого объекта.

**Способ 2.** Чтобы удалить сотрудника:

2. Перейдите в карточку интересующего вас сотрудника;
3. Справа наверху (возле кнопки **"Сохранить"**) нажмите на "3 точки";

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjectsNew/DeletedObjectsNew12.png)

4. Выберите причину бана;
5. Нажмите на кнопку "Сохранить".

## Почему нельзя удалить все сущности навсегда

В системе HubEx не предусмотрено прямое удаление элементов справочников **Объекты**, **Сотрудники**, **Заказчики**, **Компании**, а также не предусмотрено прямое удаление **Заявок**.

Сделано это по нескольким причинам:

* Между перечисленными выше справочниками и **Заявкой** существует неразрывная взаимосвязь. Заполняя **Заявку**, вы выбираете **Объект**, указываете исполнителя (из справочника **Сотрудники**), выбираете **Компанию-заказчика**.

  При создании **Заявки** автоматически заполняется поле **Инициатор заявки** - это тот пользователь, который создал **Заявку**. Инициатором могут выступать как наши **Сотрудники**, так и **Заказчики**.

  Заявка может ссылаться на другие - дочерние или родительские заявки.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjects/Ticket.jpg)

  В карточке каждого **Объекта** находится вкладка **История обслуживания**, которая представляет собой историю **Заявок** по данному объекту/оборудованию. При прямом удалении **Заявок** вы потеряете полную картину проведенных работ.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjects/Object.jpg)

  Поэтому в связи с наличием ссылочных связей между сущностями в системе прямое удаление любой из них привело бы к потере важных данных без возможности их восстановления.
* Всегда существует вероятность, что удаленные данные могут понадобиться вам в будущем. При прямом удалении восстановить их будет невозможно. Поэтому с помощью определенных фильтров по спискам, вы всегда легко можете просматривать списки удаленных **Объектов**, **Сотрудников**, **Заказчиков**, **Компаний**, и **Заявок** и при необходимости создать их заново.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjects/Filter.jpg)

## Как скрыть удаленные сущности

С помощью настройки ролей можно скрыть отображение удаленных элементов справочников: **Компании**, **Объекты|Оборудование**, **Роли**, **Пользователи (сотрудники/заказчики)**, **Файлы**.

Перейдите в консоль администратора и откройте меню **Администрирование - Роли**. Подробнее читайте в статьях: [Вход в консоль администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.txt) и [Настройка ролей](https://wiki.hubex.ru/docs/FAQ/RU/admin/Roles.html.txt).

Откройте роль, для которой вы хотите настроить полномочия.

Далее в панели отбора справа выберите (щелчком левой кнопки мыши) блоки **Компании**, **Объекты**, **Пользователи и контакты**.

В поиске по полномочиям наберите слово **"удаленные"**, чтобы сразу отфильтровать нужные полномочия.

В списке полномочий отключите пункты **Удаленные компании**, **Удаленные объекты**, **Удаленные пользователи**, **Удаленные роли** и нажмите кнопку **Сохранить** в верхнем правом углу.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/DeletedObjects/Role.jpg)
