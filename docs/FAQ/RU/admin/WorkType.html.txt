## Что такое виды работ в HubEx

**Вид работ** – это специфический вид деятельности (услуг), который может выполняться **Сотрудником** и быть направлен на **Объект**.

Например, видами работ могут быть: услуги клининга, ремонт, диагностика, услуга аренды недвижимости, услуги доставки и т.д.

**Вид работ** позволяет классифицировать **Объекты**, **Сотрудников** и **Заявки** по видам выполняемых работ. Например, отделить заявки на клининг от заявок на ремонт оборудования.

## Создание новых видов работ

Создать новый **Вид работ** можно в [Консоли администратора](https://wiki.hubex.ru/docs/FAQ/RU/admin/HowToEnterTheAdmin.html.txt) в разделе левого бокового меню **"Настройки заявки" - "Виды работ"** по кнопке **"Создать вид работ"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/WorkType.jpg)

Рассмотрим заполнение карточки **Вида работ**:

* **Название вида работ**: заполняется в свободной форме;
* **Описание**: вы можете заполнить содержание выполняемых работ;
* **Чек-лист выполняемых работ**: можно указать чек-лист, который автоматически будет подтягиваться в **Заявку** при выборе данного **Вида работ**. Чек-листы создаются в пользовательском меню системы **Чек-листы**. Подробнее читайте в статье: [Чек-листы](https://wiki.hubex.ru/docs/FAQ/RU/user/Checklists.html.txt);
* **Плановые трудозатраты**: устанавливаются, если вы хотите автоматический расчет срока окончания работ на форме выбора исполнителя в **Заявке**;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Engineer.jpg)

* **Время закрытия заявки**:установится автоматически после сохранения **Заявки** с выбранным **Видом работ**. **Крайний срок закрытия** также может быть установлен в настройке **Типа заявки**. При создании заявки система выберет наименьшее значение срока закрытия из установленного в **Типе заявке** и **Виде работ**. Подробнее читайте в статье: [Типы заявок](https://wiki.hubex.ru/docs/FAQ/RU/admin/TicketType.html.txt)

  **Крайний срок закрытия** также может быть рассчитан с помощью учета SLA в разрезе заказчиков и с учетом различных параметров **Заявки** и создания правил расчета крайнего срока закрытия. Подробнее читайте в статье: [SLA - Правила расчета крайнего срока закрытия заявки](https://wiki.hubex.ru/docs/FAQ/RU/admin/SLA.html.txt);

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Ticket.jpg)

* **Основной вид работ**: будет подставляться по умолчанию при создании нового **Объекта**;
* **Опубликован**: будет доступен пользователям для работы в системе. **Опубликованный вид работ** можно будет указать у **Объекта**, **Сотрудника** и выбрать в **Заявке**.
* **Стоимость вида работ**: используется для формирования итоговой суммы по **Заявке** в **Акте выполненных работ**. **Стоимость вида работ** будет автоматически заполняться при создании выполненных работ в **Заявке** на вкладке **Выполнение**, при необходимости ее можно изменить вручную. Подробнее читайте в статье: [Акт выполненных работ](https://wiki.hubex.ru/docs/FAQ/RU/user/ActOFAcceptance.html.txt).

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Works.jpg)

**Дополнительно!** Если добавить созданные виды работ в карточку сотрудника (исполнителя), то можно настроить **Правила автоназначения исполнителя**. Система будет подбирать исполнителя по совпадению **Вида работ** в карточке **Сотрудника** и выбранного **Вида работ** в **Заявке**. Подробнее читайте в статьях: [Правила автоназначения исполнителя](https://wiki.hubex.ru/docs/FAQ/RU/admin/RulesOfChoice.html.txt), [Автоназначение на ближайшего исполнителя](https://wiki.hubex.ru/docs/FAQ/RU/user/RulesOfChoiceGEO.html.txt).

## Виды работ в заявках

Чтобы в **Заявке** выбрать один из **Видов работ** по **Объекту**, необходимо, чтобы **Виды работ** были указаны в карточке этого **Объекта** и добавлены к **Типу заявки**.

При создании нового **Вида работ** во всех уже созданных в системе **Объектах** этот вид работ **НЕ добавится** автоматически.

Создать новый вид работ можно через форму заявки:

* При создании **Заявки**, в поле выбора **Вида работ** создать новый можно по кнопке **"+"** или **"Создать +"**;

  ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Ticket2.jpg)
* Откроется боковая форма **Вида работ** с основными полями. После заполнения всех необходимых полей нажмите кнопку **"Создать"**.

  ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Ticket3.jpg)

## Виды работ в объектах/оборудовании

При создании или редактировании **Объекта** на вкладке **"Обслуживание"** можно также создать новый **Вид работ**: нажмите кнопку **"Добавить"**, чтобы открыть список доступных **Видов работ**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Object.jpg)

Внизу списка нажмите кнопку **"+ Добавить"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Object2.jpg)

Откроется боковая форма **Вида работ** с основными полями. После заполнения всех необходимых полей нажмите кнопку **"Создать"**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkType/Object3.jpg)

## Виды работ в сотрудниках

Новые виды работ нет возможности создать через карточку сотрудника.

В карточку сотрудника можно добавить вручную только ранее созданные виды работ.

После создания нового вида работ, новый вид работ автоматически **НЕ привяжется** к карточкам уже существующих сотрудников. Новый вид работ нужно привязать к сотрудникам вручную, если это необходимо.
