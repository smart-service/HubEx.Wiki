#### Дополнительные поля для выполненных работ: обновление от 06.03.2023

Настройка дополнительных полей теперь доступна не только для объектов и заявок, но и для выполненных работ!

Дополнительные поля - это гибкие и легко конфигурируемые поля для удовлетворения потребностей любой компании.

С их помощью исполнители смогут вносить важную информацию наиболее эффективным способом.

Например, можно создать поле для описания внешнего вида объекта, заполнения даты следующей проверки и т.д.

##### Дополнительные поля для выполненных работ

Дополнительные поля создаются в консоли администратора в меню **Настройки системы - Дополнительные поля**по кнопке **Создать поле**.

Для дополнительного поля необходимо указать название и его тип: строка, число, выпадающий список и т.д. Тип задается исходя из тех данных, которые вы планируете вносить в это поле.

В значении поля **Используется для** необходимо указать для каких сущностей системы оно будет использоваться. В нашем примере - **Выполненные работы**.

Например, создаем дополнительное поле для описания внешнего вида объекта.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsWorks/Addfields.jpg)

При добавлении выполненной работы в web-версии HubEx дополнительные поля будут располагаться над полем **Комментарий к выполненной работе**.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsWorks/Addfields2.jpg)

В мобильном приложении **HubEx исполнитель** дополнительные поля будут доступны на вкладке (**Шаг 2**) **Дополнительно**. Переключаться между вкладками выполненной работы можно по кнопке внизу.

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/AdditionalFieldsWorks/Mob.jpg)

Подробнее читайте в статье базы знаний: [Создание дополнительных полей](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/AdditionalFields.html.txt).
