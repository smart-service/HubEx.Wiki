#### Список объектов и оборудования, история обслуживания объектов

В этом разделе вы узнаете:

* [Как открыть список Объектов (Оборудования) в мобильном приложении;](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/ObjectListInMob.html.txt#list)
* [Как посмотреть паспорт объекта и историю обслуживания;](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/ObjectListInMob.html.txt#history)
* [Как подать Заявку.](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/ObjectListInMob.html.txt#ticket)

В мобильном приложении **HubEx исполнитель** для пользователя доступен список **Объектов**
и **Оборудования**.

Доступ к **Объектам**
регулируется настройками **Роли** пользователя в консоли администрирования. Подробнее читайте в статье:
Настройка ролей.

Список **Объектов** позволяет получить информацию об обслуживаемом оборудовании, просмотреть историю его
обслуживания, а также быстро
создать Заявку по одному из **Объектов**.

При нажатии на любой из **Объектов** списка откроется паспорт объекта.

Паспорт объекта - это данные Объекта (Оборудования), которые отражаются в мобильном приложении HubEx исполнитель и
HubEx заказчик при сканировании QR-кода по кнопке
подачи заявки (на главном экране в правом нижнем углу). Также паспорт объекта отображается при сканировании QR-кода
через камеру телефона незарегистрированными в системе пользователями (без использования мобильных приложений HubEx,
паспорт открывается в мобильном браузере).

##### Список Объектов (Оборудования) в мобильном приложении

Чтобы открыть список объектов, перейдите в главное меню **HubEx исполнитель** (три полоски в левом
верхнем углу) и откройте пункт **Объекты|Оборудование**.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ObjectListInMob/ObjectinMob.jpg)

В списке будут отображаться доступные пользователю **Объекты**.

По щелчку на **Объект** откроется паспорт объекта. В паспорте представлены основные
характеристики
**Объекта**.

В паспорте объекта пользователь может изменить некоторые характеристики: **Класс
оборудования**, **Серийный номер**, **Описание** и **Файлы**.
Подробнее читайте в статье:
[Редактирование данных Объекта (Оборудования)](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/ObjectEditing.html.txt).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ObjectListInMob/ObjectinMob2.jpg)

Из паспорта объекта по кнопке **Посмотреть историю обслуживания** открывается история
**Заявок** по **Объекту**. В каждую Заявку можно зайти и ознакомиться с ее выполнением.
Чтобы вернуться назад к паспорту, нажмите стрелку в левом верхнем углу.

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ObjectListInMob/ObjectinMob3.jpg)

Также из паспорта объекта доступно создание **Заявки** по кнопке **Подать
заявку**.

В открывшейся форме необходимо заполнить поля: **Тип заявки**, **Вид работ**, **Исполнитель**,
**Описание заявки** и нажать кнопку **Создать заявку**. Подробнее о назначении и заполнении формы читайте в статье: [Создание заявки](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/CreatingTicket.html.txt).

![](https://wiki.hubex.ru/attachments/images/FAQ/USER/ObjectListInMob/ObjectinMob4.jpg)
