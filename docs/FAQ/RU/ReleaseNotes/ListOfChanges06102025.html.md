## Обновление Новые возможности фильтрации: поля-переключатели в заявках и Новые настройки отслеживания местоположения для мобильных сотрудников от 06.10.2025

## 🎛️ Новые возможности фильтрации: поля-переключатели в заявках

Уважаемые пользователи HubEx!

Мы продолжаем улучшать инструменты фильтрации, чтобы вы могли еще эффективнее работать с заявками. Теперь в фильтрах доступны поля с типом "Переключатель"!

### 🚀 Что нового?

**Добавлена поддержка дополнительных полей с типом "переключатель" в фильтрах:**

* Теперь можно фильтровать заявки по дополнительным полям с типом "Переключатель"
* В окне выбора фильтров появились соответствующие поля
* Простой и интуитивно понятный интерфейс: выбор "Да"/"Нет"/"Все варианты"

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges06102025/ccZVRZrV_image.png)

### 💡 Примеры использования:

1. **Фильтр "Выдано оборудование на подмену"**
   * "Да" - показать заявки, где оборудование выдано на подмену (галочка в заявке в поле включена)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges06102025/BKBzPbnz_image.png)

* "Нет" - показать заявки без выдачи оборудования на подмену (галочки в заявке в поле выключена)

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges06102025/Pv5LxM25_image.png)

1. **Быстрая сортировка по важным критериям**
   * Срочные заявки
   * Требует согласования
   * Специальные условия и др.

### 🎯 Преимущества:

✅ **Экономия времени** - мгновенная фильтрация по ключевым параметрам
✅ **Удобство использования** - простой интерфейс "Да/Нет"
✅ **Гибкость** - комбинируйте с другими фильтрами для точных результатов

### 📌 Как это работает?

1. Откройте фильтры в разделе заявок
2. Найдите нужное поле с типом "Переключатель"
3. Выберите значение: "Да" или "Нет"
4. Примените фильтр - система мгновенно отобразит подходящие заявки

## 📍 Новые настройки отслеживания местоположения для мобильных сотрудников

Уважаемые пользователи HubEx!

Мы добавили больше гибкости в настройках отслеживания местоположения мобильных сотрудников. Теперь вы можете управлять сбором координат в соответствии с потребностями вашей компании.

### 🚀 Что изменилось?

**Новая возможность в профиле пользователя:**

* Теперь можно активировать статус **"Мобильный сотрудник" (требуется для автоназначения заявок)**, при этом отключив опцию **"Отслеживать местоположение сотрудника"**
* Ранее система не позволяла сохранять такие настройки

### ⚙️ Как это работает?

**Три режима отслеживания:**

1. **Всегда** - обязательный сбор координат пользователя в МП
2. **С разрешения пользователя** - ограниченный сбор координат, если пользователь разрешил в МП доступ к данным
3. **Выключить отслеживание местосположения сотрудника** - координаты не собираются, при этом автоназначение работает

![](https://wiki.hubex.ru/attachments/images/FAQ/RELEASENOTES/ListOfChanges06102025/mV7h6jw2_image.png)

### 🎯 Что это дает?

**Для сотрудников:**

* Оптимизация энергопотребления при работе с приложением

**Для администраторов:**

* Возможность сохранения автоназначения для мобильных сотрудников, без сбора координат пользователя
* Соответствие политикам компании по защите данных

**Для системы:**

* Сотрудники с отключенным отслеживанием не участвуют в подборе по радиусу
* Обычное автоназначение заявок продолжает работать
* Автоназначение с учетом удаленности игнорирует сотрудников с выключенным отслеживанием местоположения

### 📌 Как настроить?

1. Откройте карточку пользователя в веб-версии
2. Включите опцию **"Мобильный сотрудник"**
3. Выберите нужный режим отслеживания местоположения
4. Сохраните изменения

---

💡 **Важно знать:**

* Настройка влияет только на автоназначение по радиусу
* Ручное назначение заявок остается без изменений
