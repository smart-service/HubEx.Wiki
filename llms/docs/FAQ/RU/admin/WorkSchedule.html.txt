#### Настройка графиков работы

В этом разделе вы узнаете:

* [Как создавать или редактировать графики работы](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/WorkSchedule.html.txt#schedule);
* [Как создать недельный график](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/WorkSchedule.html.txt#week).
* [Как создать сменный график](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/WorkSchedule.html.txt#shift).

**График работы** исполнителей позволяет учитывать рабочее время сотрудников при
назначении
на **Заявки**. Также
**График работы** можно использовать при настройке автораспределения исполнителей на
**Заявки**. Автоматическое распределение позволяет находить и назначать на **Заявку**
подходящего исполнителя по
определенным параметрам: **Графику работы**, **Виду работ**, особым
**Навыкам** и тд. Подробнее об этом читайте в статье:
[Правила автоназначения исполнителя](https://wiki.hubex.ru/llms/docs/FAQ/RU/admin/RulesOfChoice.html.txt), [Автоназначение на ближайшего
исполнителя](https://wiki.hubex.ru/llms/docs/FAQ/RU/user/RulesOfChoiceGEO.html.txt).

В HubEx по умолчанию создано несколько графиков работы, которые можно указывать в карточке
**Сотрудника** в web-приложении (в меню **Пользователи - Сотрудники**).

Изменять графики работы, созданные по умолчанию, нельзя, но можно создавать и настраивать свои персональные графики и
использовать их в работе.

Создание и редактирование персональных графиков находится в консоли администрирования в меню **Структура
предприятия -
Графики работы**.

Создать новый график можно по кнопке **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule.jpg)

В системе предусмотрено создание и настройка двух типов графиков:

* Недельный: включает в себя настройку рабочего времени на каждый из семи дней недели. Такой тип графика подойдет
  для создания, например, пятидневной или шестидневной рабочей недели с двумя или одним выходным днем.
* Сменный: подразумевает создание повторяющегося интервала смен и выходных дней. Этот тип подходит для создания
  таких графиков работы, как 2/2,1/3 и т.д.

Рассмотрим создание каждого типа графиков подробно.

##### Создание недельного графика работы

Для создания недельного графика выполните следующие шаги:

1. введите **Наименование**: для удобства наименование следует задавать исходя из типа графика
   и времени работы. Например, для графика работы со вторника по субботу с 10.00 до 19.00 и двумя выходными в
   воскресенье можно ввести наименование Вторник-суббота с 10 до 19;
2. с помощью флажков укажите, какие дни недели будут являться рабочими по графику, и установите время работы в
   формате чч:мм. Для текущего примера с графиком работы со вторника по субботу с 10.00 до 19.00 и двумя выходными
   в
   воскресенье и
   понедельник необходимо установить флажки для вторника, среды, четверга, пятницы, субботы, а для воскресенья и
   понедельника флажки не ставить, так как это нерабочие дни;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule2.jpg)

3. установите дату начала действия текущего графика с помощью флажка **Назначить дату начала
   календаря**: этот пункт не является обзательным для выполнения, если дату начала не задать, то по
   умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать с 20 февраля 2024 года;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule22.jpg)

4. проверьте отображение графика по кнопке **Показать график в календаре**: в новом окне
   появится распределение графика работы по месяцам. При внесении изменений в график кнопку **Показать график
   в календаре** нужно нажимать повторно;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule3.jpg)

5. нажмите кнопку **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule4.jpg)

Чтобы использовать автоназначение на заявки по графику и отслеживать рабочее время исполнителей по календарю,
необходимо указать созданный график в карточке **Сотрудника**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule5.jpg)

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule6.jpg)

##### Создание сменного графика работы

Для создания сменного графика выполните следующие шаги:

1. введите **Наименование**: для удобства наименование следует задавать исходя из типа графика
   и времени работы. Например, для сменного графика работы два дня рабочих с 10 до 22 и два дня выходных можно
   ввести наименование 2/2 с 8.30 до 20;
2. по умолчанию для настройки сменного графика на форме представлен интервал
   2/2: две рабочих смены с 10:00 до 22:00 и два выходных дня. Такой интервал будет повторяться друг за другом.
   Чтобы изменить настройки по умолчанию удаляйте строки смен и (или) выходных дней и добавляйте новые по кнопке +
   **Добавить**. Для каждой строки можно изменить тип дня графика: **Смена** или **Выходной**.

   Для нашего примера графика 2/2 поправим только время начала и окончания смен с 8.30 до 20.00.

   Обратите внимание! Время одной смены не может быть более 24 часов. Если ваши сотрудники работают более 24
   часов в смене, создайте подряд две рабочие смены на 24 часа и на дополнительное время.

   Продолжительность смены рассчиывается исходя из установленного времени и отображается под каждой строкой
   смены.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule7.png)

3. установите дату начала действия текущего графика с помощью флажка **Назначить дату начала
   календаря**: этот пункт не является обзательным для выполнения, если дату начала не задать, то по
   умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать
   с 23 мая 2024 года;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule8.png)

4. проверьте отображение графика по кнопке **Показать график в календаре**: в новом окне
   появится распределение графика работы по месяцам. При внесении изменений в график кнопку **Показать график
   в календаре** нужно нажимать повторно;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule9.png)

5. нажмите кнопку **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule10.png)

Рассмотрим еще один пример создания сменных графиков: создание сменного графика 3/2/3, где первые 3 дня это дневные
смены с 6:00 до 16:00, следом 2 дня ночные
смены с 16:00 до 2:00 и 3 выходных дня.

Для создания графика 3/2/3 выполните следующие шаги:

1. введите **Наименование**: 3/2/3 с 6 до 16, с 16 до 2;
2. как описано выше, по умолчанию для настройки сменного графика на форме представлен интервал
   2/2. Интервал по умолчанию необходимо изменить под требования примера 3/2/3 следующим образом:
   * в первой строке для типа дня **Смена** измените
     количество на 3 и установить время начала 6:00 и время
     окончания смены на 16:00;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule11.png)

   * так как следующая часть интервала - это ночные смены, то для второй строки интервала с типом дня
     **Выходной** измените тип на **Смена**, задайте количество 2 и установите время
     с 16:00 до 2:00.

     Обратите внимание! При задании ночных смен вводите время окончания следующего дня (например, 2 часа ночи), система
     автоматически рассчитает время продолжительности смены с переходом на следующий день и отобразит
     значение под строкой смены;

   ![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule12.png)

   * по кнопке **+Добавить** согласно графику добавьте строку с типом **Выходной** и количеством 3;
3. установите дату начала действия текущего графика с помощью флажка **Назначить дату начала
   календаря**: этот пункт не является обзательным для выполнения, если дату начала не задать, то по
   умолчанию график будет работать со дня его создания в системе. Например, график работы начнет действовать
   с 23 мая 2024 года;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule13.png)

4. проверьте отображение графика по кнопке **Показать график в календаре**: в новом окне
   появится распределение графика работы повторяющегося интервала 3/2/3;

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule14.png)

5. нажмите кнопку **Создать график**.

![](https://wiki.hubex.ru/attachments/images/FAQ/ADMIN/WorkSchedule/Schedule15.png)
